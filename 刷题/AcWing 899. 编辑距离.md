# 题目

https://www.acwing.com/problem/content/901/

给定 $n$ 个长度不超过 $10$ 的字符串以及 $m$ 次询问。  
每次询问给出一个字符串 $q$ 和一个整数 $k$，表示操作次数上限。

对于每次询问，求出给定的 $n$ 个字符串中有多少个字符串，可以**在最多 $k$ 次操作**内变成 $q$。  
每个字符的插入、删除或替换算作一次操作。

**输入格式**

第一行包含两个整数 $n$ 和 $m$。

接下来 $n$ 行，每行一个字符串，表示给定的字符串。

再接下来 $m$ 行，每行一个字符串和一个整数，表示一次询问。

所有字符串只包含小写字母，且长度均不超过 $10$。

**输出格式**

输出共 $m$ 行，每行输出一个整数，表示满足该询问的字符串个数。

**数据范围**

$$1 \le n, m \le 1000$$
**输入样例**

```
3 2
abc
acd
bcd
ab 1
acbd 2
```

**输出样例**

```
1
3
```

# 思路

[[AcWing 902. 最短编辑距离#思路]]
判断n次就可以.

# 代码

```c++
// 使用万能头文件
#include<bits/stdc++.h>
using namespace std;

// 定义常量N，用于数组大小
const int N = 1010;
// 存储n个输入字符串
char strs[N][N];
// n: 输入字符串数量, m: 查询次数, dp: 用于计算编辑距离的动态规划数组
int n, m, dp[N][N];
// 变量cnt在代码中未使用，可忽略
int cnt;

// 计算两个字符串之间的编辑距离
int distance(char a[], char b[]){
    // 计算两个字符串长度（从索引1开始，符合代码中字符串的存储方式）
    int la = strlen(a+1);
    int lb = strlen(b+1);
    
    // 初始化dp数组
    for(int i = 0; i <= la; i++) dp[i][0] = i;  // 将a的前i个字符变为空串需要i次删除
    for(int j = 0; j <= lb; j++) dp[j] = j;  // 将空串变为b的前j个字符需要j次插入
    
    // 动态规划计算编辑距离
    for(int i = 1; i <= la; i++){
        for(int j = 1; j <= lb; j++){
            // 如果当前字符相同，则不需要操作，继承之前的结果
            if(a[i] == b[j])
                dp[i][j] = dp[i-1][j-1];
            else
                // 如果字符不同，考虑三种操作并取最小值：
                // 1. dp[i-1][j-1]+1: 替换操作
                // 2. dp[i-1][j]+1: 删除a[i]（或视为在b中插入）
                // 3. dp[i][j-1]+1: 删除b[j]（或视为在a中插入）
                dp[i][j] = min(dp[i-1][j-1]+1, min(dp[i-1][j]+1, dp[i][j-1]+1));
        }
    }
    
    return dp[la][lb]; // 返回将a变为b所需的最小操作次数
}

int main(){
    // 输入字符串数量n和查询次数m
    cin >> n >> m;
    
    // 输入n个字符串，从索引1开始存储（方便后续计算）
    for(int i = 1; i <= n; i++)
        cin >> strs[i] + 1;  // +1表示从数组索引1开始接收字符串
    
    // 处理m次查询
    for(int i = 1; i <= m; i++){
        int limit;           // 本次查询允许的最大操作次数
        int res = 0;         // 统计满足条件的字符串数量
        char request[N];     // 存储查询字符串
        
        // 输入查询字符串和限制次数
        cin >> request + 1 >> limit;  // 同样从索引1开始存储
        
        // 遍历所有输入字符串，计算与查询字符串的编辑距离
        for(int j = 1; j <= n; j++){
            // 如果编辑距离小于等于限制次数，则计数
            if(distance(strs[j], request) <= limit)
                res++;
        }
        
        // 输出本次查询的结果
        cout << res << endl;
    }
}
```
