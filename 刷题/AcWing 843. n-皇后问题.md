# 题目

皇后问题是指将 `n` 个皇后放在 `n×n` 的国际象棋棋盘上，使得皇后不能相互攻击到，即任意两个皇后都不能处于同一行、同一列或同一斜线上。

![[Pasted image 20260205233059.png]]

现在给定整数 `n`，请你输出所有的满足条件的棋子摆法。

输入格式

共一行，包含整数 n。

输出格式

每个解决方案占 `n` 行，每行输出一个长度为 `n` 的字符串，用来表示完整的棋盘状态。

- 其中 `.` 表示某一个位置的方格状态为空。
- `Q` 表示某一个位置的方格上摆着皇后。

每个方案输出完成后，输出一个空行。

**注意**：行末不能有多余空格。

输出方案的顺序任意，只要不重复且没有遗漏即可。

数据范围
$1 \leq n \leq 9$

输入样例：

```

4

```

输出样例：

```

.Q..  
...Q  
Q...  
..Q.

..Q.  
Q...  
...Q  
.Q..
```

# 思路

https://www.acwing.com/solution/content/30231/

![[Pasted image 20260206163026.png]]

每一行必定有一个皇后，对行进行深度遍历。

对于第 r 行的第 i 个位置，判断每个点是否可以放皇后，如果可以，则放皇后，然后处理 r + 1 行。

直到 r = n，程序指行完毕。

核心思路：深度优先遍历

函数名：void dfs(int r): 深度优先遍历函数。参数r：从第r行开始放棋子，处理第r行。

递归结束判定：见代码，当 r > n的时候，说明应该处理第 n行了，也代表第 1~n 行放好棋子，也就是整个棋盘放好了棋子，也就是得到了一种解，也就是递归结束。

第r行，第i列能不能放棋子：用数组dg udg cor 分别表示：点对应的两个斜线以及列上是否有皇后。

dg[i + r] 表示 r行i列处，所在的对角线上有没有棋子，udg[n - i + r]表示 r行i列处，所在的反对角线上有没有棋子，cor[i]表示第i列上有没有棋子。如果 r行i列的对角线，反对角线上都没有棋子，即!cor[i] && !dg[i + r] && !udg[n - i + r]为真，则代表 r行i列处可以放棋子。

证明：为什么同一对角线有 $x+y$ 恒定和 $x-y$ 恒定的特征

1. “主对角线”方向：$x - y$ 恒定

在网格系统中，从左上到右下的对角线（或者平行于这条线的直线）具有 $x - y = C$（常数）的特征。

证明：

如果我们观察斜率为 $1$ 的直线，其标准方程为：

$$y = x + b$$

将其变形，可得：

$$x - y = -b$$

由于 在同一条直线上 $b$（截距）是一个定值，那么该直线上任意一点 $(x, y)$ 的 **$x - y$ 之差必然恒定**。

- **直观理解**：当你向右移动一格（$x+1$），同时向上移动一格（$y+1$），两者的差值抵消，保持不变。

2. “副对角线”方向：$x + y$ 恒定

从右上到左下的对角线具有 $x + y = C$（常数）的特征。

证明：

这条线的斜率为 $-1$，其直线方程为：

$$y = -x + b$$

将其变形，可得：

$$x + y = b$$

同理，因为 $b$ 是定值，所以该直线上所有点的 **$x, y$ 坐标之和恒等于截距 $b$**。

- **直观理解**：当你向右移动一格（$x+1$），为了留在这条线上，你必须向下移动一格（$y-1$），两者的和保持不变。

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 10;               // N 常量，棋盘最大大小为 10×10（根据需要可以调整更大）
int n;                           // n：皇后个数，棋盘大小为 n×n，从输入中获取

// 三个状态标记数组
int col[N];      // 列状态标记 col[i] 表示第 i 列是否已有皇后
int dg[2 * N];   // 对角线（右下→左上）标记 dg[i] 表示和值为 i 的对角线（即 y+x = 常数 i）
int udg[2 * N];  // 反对角线（右上→左下）标记 udg[i] 表示差值为 i 的对角线（即 y-x 统一变换后）
char g[N][N];     // 存储棋盘 g[i][j] 中为 '.' 或 'Q'，1-index 从 (1,1) 到 (n,n) 为棋盘

/**
 * 深度优先搜索，逐行尝试放置皇后
 * @param r 当前正在尝试放置的行（从 1 开始）
 */
void dfs(int r) {
    // 递归终止条件：所有行都处理完了（r > n）
    if (r > n) {
        // 找到一个合法方案，输出整个棋盘
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= n; j++) {
                cout << g[i][j];
            }
            cout << endl;         // 每行结束换行
        }
        cout << endl;             // 输出一个方案后空行
        return;
    }

    // 遍历当前行的每一列
    for (int c = 1; c <= n; c++) {
        // 检查该位置是否安全：
        // !col[c]            → 第 c 列没有皇后
        // !dg[r + c]         → 左下到右上的对角线没有皇后（r+c 为常数）
        // !udg[n + r - c]    → 左上到右下的对角线没有皇后（r-c 可能为负数，加 n 偏移以使用数组下标）
        if (!col[c] && !dg[r + c] && !udg[n + r - c]) {
            // 放置皇后，更新状态
            col[c] = 1;                 // 标记第 c 列已被占用
            dg[r + c] = 1;              // 标记下对角线（y+x=常数）已被占用
            udg[n + r - c] = 1;         // 标记上对角线（y-x=常数，偏移 n 保证非负索引）已被占用
            g[r][c] = 'Q';              // 棋盘相应位置放 'Q'

            // 递归处理下一行
            dfs(r + 1);

            // 回溯：恢复现场，供后续尝试其他列
            col[c] = 0;
            dg[r + c] = 0;
            udg[n + r - c] = 0;
            g[r][c] = '.';
        }
    }
}

int main() {
    // 输入棋盘大小（皇后个数）
    cin >> n;

    // 初始化棋盘为全空（'.'）
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            g[i][j] = '.';
        }
    }

    // 从第一行开始深度优先搜索
    dfs(1);

    return 0;
}
```
