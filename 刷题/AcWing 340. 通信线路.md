# 题目

https://www.acwing.com/activity/content/problem/content/1500/

在郊区有 N 座通信基站，P 条 **双向** 电缆，第 i 条电缆连接基站 $A_i$ 和 $B_i$。

特别地，1 号基站是通信公司的总站，N 号基站位于一座农场中。

现在，农场主希望对通信线路进行升级，其中升级第 i 条电缆需要花费 $L_i$。

电话公司正在举行优惠活动。

农场主可以指定一条从 1 号基站到 N 号基站的路径，并指定路径上不超过 K 条电缆，由电话公司免费提供升级服务。

农场主只需要支付在该路径上剩余的电缆中，升级价格最贵的那条电缆的花费即可。

求至少用多少钱可以完成升级。

#### 输入格式

第 1 行：三个整数 N，P，K。

第 2..P+1 行：第 i+1 行包含三个整数 Ai,Bi,Li。

#### 输出格式

包含一个整数表示最少花费。

若 1 号基站与 N 号基站之间不存在路径，则输出 −1。

#### 数据范围

0≤K<N≤1000,  
1≤P≤10000,  
1≤Li≤1000000

#### 输入样例：

```
5 7 1
1 2 5
3 1 4
2 4 8
3 2 3
5 2 9
3 4 7
4 5 6
```

#### 输出样例：

```
4
```

# 思路

这么抽象吗!!!!

下面把你刚刚问的几个关键问题 **系统整合在一起**，从算法整体到细节全部串起来。这样你以后再遇到这种题基本就能一眼看懂。

---

## 一、这道题整体在做什么

题目目标其实是：

> 找一条从 **1 到 n** 的路径，使得  
> **权值大于某个值的边最多使用 k 条**，  
> 并且这个 **权值尽量小**。

换句话说：

```text
允许最多 k 条“很大的边”
求这些“大边”的最小阈值
```

解决思路：

```text
二分答案 + 0-1 BFS（最短路）
```

流程：

1. 二分一个最大允许边权 `bound`
    
2. 判断是否存在一条路径  
    **使用 ≤ k 条 “权值 > bound 的边”**
    
3. 如果可以，就缩小 `bound`
    
4. 最终找到最小可行 `bound`
    

---

## 二、check(bound) 在做什么

在 `check(bound)` 里，我们把原图 **重新解释**：

|原边权|转换后权值|
|---|---|
|≤ bound|0（好边）|
|> bound|1（坏边）|

于是：

```text
dist[i] = 从1到i使用的最少坏边数量
```

我们求的是：

```text
从1到n最少需要多少条坏边
```

最后判断：

```cpp
dist[n] <= k
```

如果成立说明：

```text
存在一条路径
坏边 ≤ k
```

---

## 三、为什么要求“最短路”

你的问题是：

> 为什么要找最短路？  
> 如果最短路没走到某条路径会不会漏答案？

答案是：**不会漏**。

因为我们求的是：

```text
坏边数量最少的路径
```

设所有路径的坏边数量：

|路径|坏边数|
|---|---|
|A|5|
|B|3|
|C|1|

最短路会找到：

```text
C：坏边 = 1
```

如果：

```text
1 ≤ k
```

说明存在合法路径。

如果最短路结果：

```text
dist[n] = 4
```

而：

```text
k = 2
```

说明：

```text
所有路径坏边 ≥ 4
```

因此：

```text
不可能存在 ≤2 的路径
```

所以 **最短路就是最强判定**。

---

## 四、为什么使用 0-1 BFS

因为转换后的边权只有：

```text
0 或 1
```

普通 Dijkstra：

```text
O(m log n)
```

而 **0-1 BFS**：

```text
O(n + m)
```

实现方式：

使用 **双端队列 deque**。

规则：

|边权|操作|
|---|---|
|0|push_front|
|1|push_back|

---

## 五、为什么好边要放队头

代码：

```cpp
if (!x) q.push_front(j);
else q.push_back(j);
```

其中：

```cpp
x = w[i] > bound
```

所以：

|边类型|操作|
|---|---|
|好边|push_front|
|坏边|push_back|

原因：

```text
好边不会增加坏边数量
```

所以它应该 **优先扩展**。

可以把队列想象成：

```text
[ 更优路径 | 较差路径 ]
```

好边：

```text
不增加代价
→ 插队到最前面
```

坏边：

```text
代价 +1
→ 排到后面
```

这样就模拟了 **Dijkstra 的优先队列效果**。

---

## 六、为什么不用普通 BFS

假设：

```
1 → 2  (坏边)
1 → 3  (好边)
3 → 4  (好边)
```

如果全部 `push_back`：

队列：

```
[2 , 3]
```

算法会先处理 **更差的路径**。

但如果使用 0-1 BFS：

```
[3 , 2]
```

优先处理：

```
1 → 3 → 4
```

这才是最优路径。

---

## 七、check() 的真实含义

`check(bound)` 实际上是在做一件事：

```text
把 ≤ bound 的边当作免费
把 > bound 的边当作代价1
```

然后计算：

```text
从1到n最少需要多少次“超限边”
```

如果：

```text
≤ k
```

说明：

```text
这个 bound 可行
```

---

## 八、为什么可以二分答案

关键性质：

```text
bound 越大
坏边越少
```

举例：

|bound|坏边数量|
|---|---|
|2|5|
|5|3|
|10|1|

随着 `bound` 增大：

```text
可行性只会从 false → true
```

不会反复变化。

所以具有：

```text
单调性
```

因此可以：

```text
二分答案
```

---

## 九、整题算法结构

完整流程：

```
二分 bound
      ↓
check(bound)
      ↓
0-1 BFS
      ↓
求最少坏边
      ↓
判断 ≤ k
```

复杂度：

```text
O((n + m) log W)
```

其中：

```
W = 边权范围
```

## 十、一句话理解整题

这题本质是：

```text
二分最大允许边权
+
0-1 BFS 计算最少超限边
```

最终找到：

```text
使得坏边 ≤ k 的最小 bound
```

---

如果你愿意，我可以再给你讲一个 **竞赛里非常经典的套路：**

> **“二分答案 + 最短路判定”**

很多 **图论最难题（NOI / ICPC）** 都是这个套路，你会发现这题其实是一个 **模板题**。

```c++
#include<bits/stdc++.h>
using namespace std;

// N：点数上限
// M：边数上限（无向图要乘2）
const int N=1010,M=20010;

int n,m,k;

// 邻接表
int h[N],e[M],ne[M],idx;

// dist：到各点需要经过的“坏边”数量
// st：是否已确定最短路
int dist[N],st[N];

// w：边权
int w[M];


// 加边（邻接表）
void add(int a,int b,int c){
    e[idx]=b;        // 边的终点
    w[idx]=c;        // 边权
    ne[idx]=h[a];    // 指向下一条边
    h[a]=idx++;      // 更新头节点
}


// 检查：当最大允许边权为 bound 时是否可行
// 即：最多经过 k 条权值 > bound 的边
bool check(int bound){

    // 初始化最短路数组
    memset(dist,0x3f,sizeof dist);
    memset(st,0,sizeof st);

    deque<int> q;

    dist[1]=0;          // 从1号点出发
    q.push_back(1);

    // 0-1 BFS
    while(!q.empty()){

        int u=q.front();
        q.pop_front();

        if(st[u]) continue;
        st[u]=1;

        // 遍历所有邻边
        for(int i=h[u];~i;i=ne[i]){

            int v=e[i];

            // 如果边权 > bound，则认为是一条“坏边”
            int x=w[i]>bound;

            // 松弛
            if(dist[v] > dist[u]+x){

                dist[v]=dist[u]+x;

                // 0权边放队头
                if(!x) q.push_front(v);
                // 1权边放队尾
                else q.push_back(v);
            }
        }
    }

    // 如果到n的坏边数量 ≤ k，则说明可行
    return dist[n] <= k;
}


int main(){

    cin>>n>>m>>k;

    // 初始化邻接表
    memset(h,-1,sizeof h);

    // 读入边
    for(int i=0;i<m;i++){
        int a,b,c;
        cin>>a>>b>>c;

        // 无向图
        add(a,b,c);
        add(b,a,c);
    }

    // 二分答案：最大允许的免费边权
    int l=0,r=1e6+1;

    while(l<r){

        int mid=(l+r)>>1;

        // 如果当前 bound 可行，尝试更小
        if(check(mid))
            r=mid;
        else
            l=mid+1;
    }

    // 如果仍然是上界，说明无解
    if(l==1e6+1)
        cout<<-1<<endl;
    else
        cout<<l<<endl;
}
```
