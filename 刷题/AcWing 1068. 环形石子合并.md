# 题目

https://www.acwing.com/problem/content/1070/

将 n 堆石子绕圆形操场排放，现要将石子有序地合并成一堆。

规定每次只能选相邻的两堆合并成新的一堆，并将新的一堆的石子数记做该次合并的得分。

请编写一个程序，读入堆数 n 及每堆的石子数，并进行如下计算：

- 选择一种合并石子的方案，使得做 n−1 次合并得分总和最大。
- 选择一种合并石子的方案，使得做 n−1 次合并得分总和最小。

#### 输入格式

第一行包含整数 n，表示共有 n 堆石子。

第二行包含 n 个整数，分别表示每堆石子的数量。

#### 输出格式

输出共两行：

第一行为合并得分总和最小值，

第二行为合并得分总和最大值。

#### 数据范围

1≤n≤200

#### 输入样例：

```
4
4 5 9 4
```

#### 输出样例：

```
43
54
```

# 思路

复习: [[AcWing 282. 石子合并]]

不同点是,这是一个环形,只要将1~n的石子存在n+1~2n就可以,然后计算i~i+n-1,i from 1 to n+1就可

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

// N 是题目给定的最大规模，因为要“破环成链”存双倍长度，所以数组开到 2*N
const int N = 210; 
int n, a[N * 2], s[N * 2], f1[N * 2][N * 2], f2[N * 2][N * 2];

int main() {
    cin >> n;
    
    // 1. 破环成链：将环形数据线性化。
    // 例如环是 [1, 2, 3]，线性化后变成 [1, 2, 3, 1, 2, 3]
    // 这样任意长度为 n 的连续子区间都代表了原环的一种合并顺序。
    for (int i = 1; i <= n; i++) {
        int x;
        cin >> x;
        a[i] = a[n + i] = x; 
    }

    // 2. 预处理前缀和：s[i] 表示前 i 堆石子的总和。
    // 后面计算区间 [i, j] 的石子总重量时，直接使用 s[j] - s[i-1]，时间复杂度 O(1)
    for (int i = 1; i <= n * 2; i++) {
        s[i] = s[i - 1] + a[i];
    }

    // 3. 初始化 DP 数组
    // f1 存最大值，全局变量默认是 0，无需额外处理
    // f2 存最小值，需要初始化为无穷大 (0x3f3f3f3f)
    memset(f2, 0x3f, sizeof f2);

    // 4. 区间 DP 主循环
    // len 表示当前处理的合并区间长度
    for (int len = 1; len <= n; len++) {
        // i 表示区间的左端点
        for (int i = 1; i + len - 1 <= n * 2; i++) {
            int j = i + len - 1; // j 是区间的右端点
            
            // 基础情况：如果区间只有一堆石子，合并代价为 0
            if (i == j) {
                f1[i][j] = f2[i][j] = 0;
            }

            // k 是分割点，将区间 [i, j] 分为 [i, k] 和 [k+1, j] 两部分进行合并
            for (int k = i; k <= j - 1; k++) {
                // 状态转移方程：
                // 当前区间的得分 = 左边部分的得分 + 右边部分的得分 + 本次合并得到的得分(区间总和)
                f1[i][j] = max(f1[i][j], f1[i][k] + f1[k + 1][j] + s[j] - s[i - 1]);
                f2[i][j] = min(f2[i][j], f2[i][k] + f2[k + 1][j] + s[j] - s[i - 1]);
            }
        }
    }

    // 5. 统计结果
    // 因为是环形，最后的结果可能是以原环中任何一个位置开始、长度为 n 的区间
    int max_s = 0, min_s = 0x3f3f3f3f;
    for (int i = 1; i <= n; i++) {
        max_s = max(max_s, f1[i][i + n - 1]); // 在 2n 长度中找最大
        min_s = min(min_s, f2[i][i + n - 1]); // 在 2n 长度中找最小
    }

    cout << min_s << endl << max_s << endl;
    return 0;
}
```
