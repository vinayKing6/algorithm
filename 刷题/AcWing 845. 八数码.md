# 题目

https://www.acwing.com/activity/content/problem/content/908/

在一个 3×3 的网格中，1∼8 这 8 个数字和一个 'x' 恰好不重不漏地分布在网格中。

游戏过程中，可以将 'x' 与其上、下、左、右四个方向之一的数字交换（如果存在）。

目标是通过交换，使网格变为正确排列：

```
1 2 3
4 5 6
7 8 x
```

## 输入格式

输入占一行，将 3×3 的初始网格描绘出来，用空格分隔。

例如，如果初始网格如下所示：

```
1 2 3 
x 4 6 
7 5 8 
```

则输入为：`1 2 3 x 4 6 7 5 8`

## 输出格式

输出占一行，包含一个整数，表示最少交换次数。

如果不存在解决方案，则输出 `-1`。

## 输入样例

```
2 3 4 1 5 x 7 6 8
```

## 输出样例

```
19
```

# 思路

https://www.acwing.com/solution/content/15149/

1、题目的目标

![[Pasted image 20260208154041.png]]

**求最小步数 -> 用BFS**

2、移动情况

![[Pasted image 20260208154048.png]]

移动方式：

![[Pasted image 20260208154056.png]]

转以后：$a = x + dx[i]$, $b = y + dy[i]$.

思想：将每一种情况作为1个节点，目标情况即为终点

从初始状况移动到目标情况 —> 求最短路

3、问题

第一点：怎么表示一种情况使其能作为节点？

第二点：如何记录每一个状态的“距离”（即需要移动的次数）？

第三点：队列怎么定义，dist数组怎么定义？

4、解决方案

将 “3*3矩阵” 转化为 “字符串”

![[Pasted image 20260208154106.png]]
​

所以：

```c++
队列可以用 queue<string>
//直接存转化后的字符串
dist数组用 unordered_map<string, int>
//将字符串和数字联系在一起，字符串表示状态，数字表示距离
```

5、矩阵与字符串的转换方式

![[Pasted image 20260208154112.png]]

提一嘴,为什么典型的bfs不能用,因为搜到最后一个点不一定是"12345678x"

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

string g="";  // 存储初始状态（9个字符的字符串）
typedef pair<int,int> PII;  // 定义坐标对

int bfs(){
    queue<string> q;  // BFS队列，存储待处理的状态
    unordered_map<string,int> d;  // 记录每个状态到初始状态的最短距离
    
    // 初始化：将初始状态加入队列，距离为0
    q.push(g);
    d[g]=0;
    
    // 定义四个移动方向：右、左、下、上
    PII pos[4]={{0,1},{0,-1},{1,0},{-1,0}};
    
    // BFS主循环
    while(!q.empty()){
        // 从队列中取出当前状态
        string state=q.front();
        q.pop();
        
        // 获取当前状态到初始状态的距离
        int dist=d[state];
        
        // 如果是目标状态，返回距离（即最小交换次数）
        if(state == "12345678x")
            return dist;
        
        // 找到'x'（空格）在当前状态中的位置（索引0-8）
        int k=state.find('x');
        
        // 尝试四个方向的移动
        for(int i=0;i<4;i++){
            // 计算'x'移动后的新坐标
            int x=k/3+pos[i].first;  // 行坐标：k/3得到行，加上方向偏移
            int y=k%3+pos[i].second; // 列坐标：k%3得到列，加上方向偏移
            
            // 检查新坐标是否在3x3网格范围内
            if(x>=0&&x<3&&y>=0&&y<3){
                // 交换'x'和目标位置的字符
                // 新位置在一维数组中的索引：x*3+y
                swap(state[k],state[x*3+y]);
            
                // 如果这是从未访问过的新状态
                if(!d.count(state)){
                    d[state]=dist+1;  // 记录新状态的距离（比当前状态多一步）
                    q.push(state);    // 将新状态加入队列待处理
                }
            
                // 恢复状态，以便尝试其他方向
                // 这是关键：在尝试下一个方向前，需要将状态恢复到原始状态
                swap(state[k],state[x*3+y]);
            }
        }
    }
    
    // 如果队列为空仍未找到目标状态，说明无解
    return -1;
}

int main(){
    // 读入3x3网格的9个字符（1-8和x）
    // 例如：2 3 4 1 5 x 7 6 8
    for(int i=0;i<9;i++){
        char c;
        cin>>c;
        g+=c;  // 将字符连接成字符串
    }
    
    // 调用BFS，输出最少交换次数
    cout<<bfs()<<endl;
    
    return 0;
}
```
