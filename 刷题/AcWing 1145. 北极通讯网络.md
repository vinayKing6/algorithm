# 题目

https://www.acwing.com/problem/content/description/1147/

  
北极的某区域共有 n 座村庄，每座村庄的坐标用一对整数 (x,y) 表示。

为了加强联系，决定在村庄之间建立通讯网络，使每两座村庄之间都可以直接或间接通讯。

通讯工具可以是无线电收发机，也可以是卫星设备。

无线电收发机有多种不同型号，不同型号的无线电收发机有一个不同的参数 d，两座村庄之间的距离如果不超过 d，就可以用该型号的无线电收发机直接通讯，d 值越大的型号价格越贵。现在要先选择某一种型号的无线电收发机，然后统一给所有村庄配备，**数量不限**，但型号都是 **相同的**。

配备卫星设备的两座村庄无论相距多远都可以直接通讯，但卫星设备是 **有限的**，只能给一部分村庄配备。

现在有 k 台卫星设备，请你编一个程序，计算出应该如何分配这 k 台卫星设备，才能使所配备的无线电收发机的 d 值最小。

例如，对于下面三座村庄：

![[Pasted image 20260313143911.png]]

其中，|AB|=10,|BC|=20,|AC|=$10\sqrt 5$≈22.36。

如果没有任何卫星设备或只有 1 台卫星设备 (k=0 或 k=1)，则满足条件的最小的 d=20，因为 A 和 B，B 和 C 可以用无线电直接通讯；而 A 和 C 可以用 B 中转实现间接通讯 (即消息从 A 传到 B，再从 B 传到 C)；

如果有 2 台卫星设备 (k=2)，则可以把这两台设备分别分配给 B 和 C ，这样最小的 d 可取 10，因为 A 和 B 之间可以用无线电直接通讯；B 和 C 之间可以用卫星直接通讯；A 和 C 可以用 B 中转实现间接通讯。

如果有 3 台卫星设备，则 A,B,C 两两之间都可以直接用卫星通讯，最小的 d 可取 0。

#### 输入格式

第一行为由空格隔开的两个整数 n,k;

接下来 n 行，每行两个整数，第 i 行的 xi,yi 表示第 i 座村庄的坐标 (xi,yi)。

#### 输出格式

一个实数，表示最小的 d 值，结果保留 2 位小数。

#### 数据范围

1≤n≤500,  
0≤x,y≤$10^4$,  
0≤k≤100

#### 输入样例：

```
3 2
10 10
10 0
30 0
```

#### 输出样例：

```
10.00
```

# 思路

什么抽象题????
有点像[[AcWing 340. 通信线路]]

## 问题本质

题目给出：

- **n 个点**
    
- 每个点有坐标 `(x, y)`
    
- 要把这些点连接成 **k 个连通块**
    

边的权值是 **两点之间的欧几里得距离**：

[  
d=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}  
]

目标是：

> **让每个连通块内部连接，并且使得使用的最大边长度最小。**

这个问题本质是：

```
K 聚类问题（K-clustering）
```

解决方法：

```
最小生成树 + 提前停止
```

---

## 图模型

把每个点看作一个节点。

任意两个点之间都可以连边：

```
完全图
```

边权：

```
两点之间的欧几里得距离
```

边数为：

[  
\frac{n(n-1)}{2}  
]

例如：

```
n = 4
```

边为：

```
1-2
1-3
1-4
2-3
2-4
3-4
```

---

## 边的结构

代码中定义了边结构：

```cpp
struct Edge
{
    int a, b;
    double w;

    bool operator<(const Edge &t) const
    {
        return w < t.w;
    }
};
```

含义：

- `a`：起点
    
- `b`：终点
    
- `w`：边权（距离）
    

并且定义了排序规则：

```
按照边长从小到大排序
```

这是 **Kruskal 算法的关键步骤**。

---

## 计算两点距离

函数：

```cpp
double get_dist(PII a, PII b)
```

计算公式：

[  
\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}  
]

代码逻辑：

```cpp
int dx = a.x - b.x;
int dy = a.y - b.y;
return sqrt(dx * dx + dy * dy);
```

---

## 构建所有边

代码：

```cpp
for (int i = 0; i < n; i ++ )
    for (int j = 0; j < i; j ++ )
        e[m ++ ] = {i, j, get_dist(q[i], q[j])};
```

含义：

枚举 **所有点对**。

例如：

```
点0
点1
点2
```

生成边：

```
0-1
0-2
1-2
```

每条边都会存入数组 `e`。

变量：

```
m = 边数
```

---

## Kruskal 最小生成树

算法流程：

1. 按边权排序
    
2. 从最小边开始尝试加入
    
3. 如果不会形成环，则加入
    
4. 使用并查集维护连通性
    

排序代码：

```cpp
sort(e, e + m);
```

---

## 并查集结构

数组：

```
p[i]
```

表示：

```
节点 i 的父节点
```

初始化：

```cpp
for (int i = 0; i < n; i ++ )
    p[i] = i;
```

表示：

```
每个点都是独立集合
```

---

## 查找集合代表

函数：

```cpp
int find(int x)
{
    if (p[x] != x) 
        p[x] = find(p[x]);
    return p[x];
}
```

作用：

```
找到集合的根节点
```

并且进行了 **路径压缩优化**。

---

## 合并集合

代码：

```cpp
int a = find(e[i].a);
int b = find(e[i].b);

if (a != b)
{
    p[a] = b;
}
```

含义：

如果两个点属于不同集合：

```
连接这条边
```

---

## 连通块数量控制

变量：

```cpp
int cnt = n;
```

表示：

```
当前连通块数量
```

每合并一次：

```
cnt --
```

普通最小生成树最终：

```
cnt = 1
```

但本题要求：

```
cnt = k
```

因此代码：

```cpp
if (cnt <= k) break;
```

当连通块数量达到 `k` 时停止。

---

## 为什么答案是最后一条边

代码：

```cpp
res = w;
```

记录：

```
最后加入的边的长度
```

原因：

当连通块从 `k+1` 变成 `k` 时，

加入的这条边就是：

```
让所有连通块保持 k 个时
所需要的最小最大边
```

因此它就是答案。

---

## 算法整体流程

程序执行步骤：

1. 输入 `n` 和 `k`
    
2. 输入所有点坐标
    
3. 计算所有点之间的距离
    
4. 生成所有边
    
5. 按边权排序
    
6. 使用 Kruskal 合并集合
    
7. 当连通块数量达到 `k` 时停止
    
8. 输出最后加入的边权
    

---

## 时间复杂度

边数：

$$  
E = \frac{n(n-1)}{2}  
$$

最大约：

```
125000
```

排序复杂度：

```
O(E log E)
```

并查集合并：

```
O(E α(n))
```

整体复杂度：

```
O(E log E)
```

完全可以接受。

---

## 核心思想总结

这题本质是：

```
最小生成树 + 提前停止
```

普通 MST：

```
一直连到 1 个连通块
```

本题：

```
只连到 k 个连通块
```

因此得到 **K 聚类的最优解**。

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

// N：最大点数
// M：完全图最大边数 n(n-1)/2
const int N=510,M=N*N/2;

int n,k;          // n 个点，最终分成 k 个连通块
int p[N];         // 并查集父节点数组
int idx;          // 当前边数量
int cnt;          // 当前连通块数量

typedef pair<int,int> PII;
PII q[N];         // 存储每个点的坐标 (x,y)


// 并查集查找函数（路径压缩）
int find(int x){
    if(p[x]!=x){
        p[x]=find(p[x]);
    }
    return p[x];
}


// 计算两点之间的欧几里得距离
double getDist(int i,int j){
    int dx=q[i].first-q[j].first;
    int dy=q[i].second-q[j].second;
    
    return sqrt(dx*dx+dy*dy);
}


// 边结构体
struct Edge{
  int a,b;     // 边的两个端点
  double c;    // 边权（距离）
};

Edge e[M];     // 存储所有边


// 排序规则：按边权从小到大
bool cmp(Edge e1,Edge e2){
    return e1.c<e2.c;
}


// Kruskal算法
double kruskal(){

    // 遍历所有边（已经排好序）
    for(int i=0;i<idx;i++){

        // 找两个端点的集合
        int pa=find(e[i].a);
        int pb=find(e[i].b);

        double c=e[i].c;

        // 如果属于不同集合
        if(pa != pb){

            // 合并集合
            p[pa]=pb;

            // 连通块数量减少
            cnt--;

            // 当连通块数量变成 k 时
            // 当前边就是答案
            if(cnt == k){
                return c;
            }

            // 特殊情况：如果要求0个（极少出现）
            if(cnt == 1 && k ==0)
                return c;
        }
        
    }

    return 0;
}


int main(){

    cin>>n>>k;

    // 初始每个点都是一个集合
    cnt=n;

    // 输入所有点坐标
    for(int i=1;i<=n;i++){
        int a,b;
        cin>>a>>b;
        q[i]={a,b};
    }

    // 构建完全图
    // 计算所有点对之间的距离
    for(int i=1;i<=n-1;i++){
        for(int j=i+1;j<=n;j++){
            e[idx++]={i,j,getDist(i,j)};
        }
    }

    // 初始化并查集
    for(int i=1;i<=n;i++)
        p[i]=i;

    // 按边权排序
    sort(e,e+idx,cmp);

    // 运行 Kruskal
    printf("%.2f",kruskal());
}
```
