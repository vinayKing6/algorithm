# 题目

https://www.acwing.com/activity/content/problem/content/1526/

发展采矿业当然首先得有矿井，小 FF 花了上次探险获得的千分之一的财富请人在岛上挖了 n 口矿井，但他似乎忘记了考虑矿井供电问题。

为了保证电力的供应，小 FF 想到了两种办法：

1. 在矿井 i 上建立一个发电站，费用为 $v_i$（发电站的输出功率可以供给任意多个矿井）。
2. 将这口矿井 i 与另外的已经有电力供应的矿井 j 之间建立电网，费用为 $p_{i,j}$。

小 FF 希望你帮他想出一个保证所有矿井电力供应的最小花费方案。

#### 输入格式

第一行包含一个整数 n，表示矿井总数。

接下来 n 行，每行一个整数，第 i 个数 vi 表示在第 i 口矿井上建立发电站的费用。

接下来为一个 n×n 的矩阵 P，其中 $p_{i,j}$ 表示在第 i 口矿井和第 j 口矿井之间建立电网的费用。

数据保证 $p_{i,j}=p_{j,i}$，且 $p_{i,i}$=0。

#### 输出格式

输出一个整数，表示让所有矿井获得充足电能的最小花费。

#### 数据范围

1≤n≤300,  
0≤$v_i,p_{i,j}$≤105

#### 输入样例：

```
4
5
4
4
3
0 2 2 2
2 0 3 3
2 3 0 4
2 3 4 0
```

#### 输出样例：

```
9
```

# 思路

模板题: [[AcWing 859. Kruskal算法求最小生成树]]
要创建一个虚拟源点
对于这道题,我们可以假设一个虚拟的”超级发电站”.  
而在每一个矿井修建发电站就相当于是从超级发电站向其连一条权值为v[i]的边.  
之后我们惊奇的发现这就是一道最小生成树的模板题.  
直接上克鲁斯卡尔算法即可..

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 310;
int n, g[N][N], p[N], idx, ans;

// 定义边结构体，用于存储起点、终点和权重
struct Edge {
    int a, b, c;
};

Edge e[N * N];

// 排序规则：按边权从小到大排序（Kruskal 核心步骤）
bool cmp(Edge e1, Edge e2) {
    return e1.c < e2.c;
}

// 并查集查找：查找节点 x 的根节点（带路径压缩）
int find(int x) {
    if (p[x] != x) {
        p[x] = find(p[x]);
    }
    return p[x];
}

// Kruskal 算法：贪心选择边来构建最小生成树
void kruaskal() {
    for (int i = 0; i < idx; i++) {
        int pa = find(e[i].a);
        int pb = find(e[i].b);
        // 如果两个节点不在同一个连通分量中，则合并
        if (pa != pb) {
            ans += e[i].c; // 累计最小成本
            p[pa] = pb;    // 合并集合
        }
    }
}

int main() {
    cin >> n;
    
    // 初始化并查集，每个节点初始都是自己的父节点
    for (int i = 0; i <= n; i++)
        p[i] = i;

    /* 技巧点：虚拟源点
       我们将“在第 i 号田打井”看作是“从虚拟节点 0 向节点 i 连一条边”。
       这样，所有的开销（打井或修路）都变成了“边”的权值。
    */
    for (int i = 1; i <= n; i++) {
        int v;
        cin >> v;
        e[idx++] = {0, i, v}; // 0号点到i号点的边权为打井成本
    }

    // 读取图中已有节点间的边权（例如修路成本）
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            cin >> g[i][j];
        }
    }

    // 将矩阵中的边信息加入边集数组
    for (int i = 1; i <= n; i++) {
        for (int j = i; j <= n; j++) { // 优化：无向图只需存一半，且避免自环
            if (g[i][j]) {
                e[idx++] = {i, j, g[i][j]};
            }
        }
    }

    // 1. 按权值升序排列所有边
    sort(e, e + idx, cmp);
    
    // 2. 执行 Kruskal 算法
    kruaskal();

    // 3. 输出最小总成本
    cout << ans << endl;

    return 0;
}
```
