# 题目

https://www.acwing.com/problem/content/897/

给定一个长度为 $N$ 的数列，求数值严格单调递增的子序列的长度最长是多少。

输入格式

第一行包含整数 $N$。

第二行包含 $N$ 个整数，表示完整序列。

输出格式

输出一个整数，表示最大长度。

数据范围

$1 ≤ N ≤ 1000$

$-10^9 ≤ \text{数列中的数} ≤ 10^9$

输入样例

```
7
3 1 2 1 8 5 6
```

输出样例

```
4
```

## 算法1

(动态规划) $O(n^2)$

状态表示：$f[i]$ 表示从第一个数字开始算，以 $w[i]$ 结尾的最大的上升序列。(以 $w[i]$ 结尾的所有上升序列中属性为最大值的那一个)

状态计算（集合划分）：$j \in (0,1,2,..,i-1)$, 在 $w[i] > w[j]$ 时，  
$f[i] = \max(f[i], f[j] + 1)$。

有一个边界，若前面没有比 $i$ 小的，$f[i]$ 为 $1$（自己为结尾）。

最后在找 $f[i]$ 的最大值。

时间复杂度

$O(n^2)$ 状态数 $(n)$ * 转移数 $(n)$

## 算法2

(动态规划 + 二分) $O(n \log n)$

状态表示：$f[i]$ 表示长度为 $i$ 的最长上升子序列，末尾最小的数字。(长度为 $i$ 的最长上升子序列所有结尾中，结尾最小 $\min$ 的) 即长度为 $i$ 的子序列末尾最小元素是什么。

状态计算：对于每一个 $w[i]$, 如果大于 $f[cnt-1]$ (下标从 0 开始，$cnt$ 长度的最长上升子序列，末尾最小的数字)，那就 $cnt+1$，使得最长上升序列长度 $+1$，当前末尾最小元素为 $w[i]$。 若 $w[i]$ 小于等于 $f[cnt-1]$, 说明不会更新当前的长度，但之前末尾的最小元素要发生变化，找到第一个 大于或等于 (这里不能是大于) $w[i]$，更新以那时候末尾的最小元素。

$f[i]$ 一定是一个单调递增的数组，所以可以用二分法来找第一个大于或等于 $w[i]$ 的数字。

时间复杂度

$O(n \log n)$ 状态数 $(n)$ * 转移数 $(\log n)$

# 代码

## 算法1

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1010;

int n, a[N], dp[N]; // a[N]存储输入的序列, dp[N]用于动态规划

int main(){
    cin >> n; // 输入序列长度
    for(int i = 1; i <= n; i++)
        cin >> a[i]; // 输入序列元素
         
    // 动态规划过程
    for(int i = 1; i <= n; i++){
        dp[i] = 1; // 以a[i]结尾的序列至少有一个数字,即他自己
        for(int j = 1; j <= i - 1; j++){ // 枚举所有可能的子序列结尾
            if(a[j] < a[i]) // 如果a[j]比a[i]小，说明可以构成上升子序列
                dp[i] = max(dp[i], dp[j] + 1); // 转移方程：dp[i] = max(dp[i], dp[j] + 1)
        }
    }
    
    int res = 0; // 存储最长上升子序列的长度
    for(int i = 1; i <= n; i++)
        res = max(res, dp[i]); // 找出所有dp[i]中的最大值
        
    cout << res << endl; // 输出结果
}
```

## 算法2

```c++
#include<bits/stdc++.h>
using namespace std;

const int N=100010;  // 定义最大数组长度
int n,a[N],f[N];     // a[]:输入序列，f[]:维护的最长上升子序列的数组

int main(){
    // 输入序列长度
    cin>>n;
    
    // 输入原始序列，下标从1开始
    for(int i=1;i<=n;i++)
        cin>>a[i];
        
    int cnt=1;           // cnt:当前最长上升子序列的长度
    f[cnt]=a[1];         // 初始化：第一个元素直接放入f数组
    
    // 遍历序列中的每个元素（从第二个开始）
    for(int i=2;i<=n;i++){
        // 情况1：当前元素大于f数组最后一个元素
        // 说明可以扩展最长上升子序列长度
        if(a[i]>f[cnt])
            f[++cnt]=a[i];
        else{
            // 情况2：当前元素不大于f数组最后一个元素
            // 使用二分查找在f数组中找到第一个大于等于a[i]的位置
            
            int l=1,r=cnt;  // 二分查找的左右边界
            while(l<=r){
                int mid=(l+r)>>1;   // 计算中间位置
                
                // 如果f[mid]大于等于a[i]，说明目标位置在mid或左边
                if(f[mid]>=a[i])
                    r=mid-1;        // 缩小右边界
                else
                    l=mid+1;        // 否则目标位置在右边
            }
            // 循环结束时，l指向第一个大于等于a[i]的位置
            f[l]=a[i];              // 用a[i]替换该位置的值
        }
    }
    
    // 输出最长上升子序列的长度
    cout<<cnt<<endl;
    
    return 0;
}
```
