# 题目描述

https://www.acwing.com/problem/content/4/

一个背包容量为 $V$，有 $N$ 种物品，第 $i$ 种物品最多有 $s_i$ 件，每件体积是 $v_i$，价值是 $w_i$。

求解将哪些物品装入背包，可使物品体积总和不超过背包容量，且价值总和最大。

输出最大价值。

输入格式

第一行两个整数 $N$ 和 $V$，用空格隔开，分别表示物品种数和背包容积。

接下来有 $N$ 行，每行三个整数 $v_i$，$w_i$，$s_i$，用空格隔开，分别表示第 $i$ 种物品的体积、价值和数量。

输出格式

输出一个整数，表示最大价值。

输入样例

```
4 5
1 2 3
2 4 1
3 4 3
4 5 2
```

输出样例

```
10
```

# 思路

## 转换成01背包问题

[[AcWing 2.01背包问题]]

**转换方法：**

1. **展开物品**  
    将第 `i` 种物品的 `s[i]` 件，**直接复制**为 `s[i]` 个独立的物品。  
    例如：若物品 A 有 3 件，每件体积为 `v`、价值为 `w`，则生成 3 个完全相同的物品（每个体积为 `v`、价值为 `w`）。
    
2. **更新数组**
    
    - **体积数组 `v_new`**：将所有生成的单件物品的体积按顺序存入。
    - **价值数组 `w_new`**：将所有生成的单件物品的价值按顺序存入。
    - **更新物品总数 `n_new`**： `n_new = sum(s[i])`（所有物品的件数之和）。
3. **套用 01 背包**  
    将 `n_new`、背包容量 `m`、`v_new`、`w_new` 作为输入，直接使用 01 背包的递推求解。  
    递推公式：
    
    ``` c++
    dp[j] = max(dp[j], dp[j - v_new[i]] + w_new[i])  
    ```
    
    其中 `dp[j]` 表示容量为 `j` 时的最大价值，需从 `m` 到 `v_new[i]` 逆序更新。

## 二进制优化

https://www.acwing.com/solution/content/20115/

在多重背包问题中，我们有 N 种物品，每种物品最多可以选 s 个。朴素的做法是：对于每种物品，遍历可选数量 k（0 到 s），更新背包。这样相当于把问题转化为 ∑s 个物品的 01 背包，复杂度 O(N * V * s)，当 s 很大时效率很低。

二进制优化的核心思路是：我们不逐一枚举 k，而是将 s 个相同的物品按 1, 2, 4, ..., 2^k, c 这样的数分组打包（c 是剩余的部分，保证总和为 s）。然后，把这些打包后的组看作新的独立“物品”，对这些新物品跑一次 01 背包。

之所以能这样分组，是因为数学上的一个重要性质：用 1, 2, 4, ..., 2^k, c 这些数的子集相加，可以组合出 0 到 s 之间的任何整数。例如，s=5 时，我们拆成 1, 2, 2。拿 0 个：不选任何组；拿 1 个：选第一个组（1）；拿 2 个：选第二个组（2）；拿 3 个：选第一组和第二组（1+2）；拿 4 个：选第二组和第三组（2+2）；拿 5 个：全选（1+2+2）。所有可能的选择数量都被覆盖了。

因此，原问题中“选择 k 个这种物品（k从0到s）”的所有可能性，都等价于“从这些二进制分组中选出若干组（对应一个子集）”。动态规划在考虑是否选取这些分组时，自然就遍历了所有可能的 k，并从中选出最优解。

这就是二进制优化正确且高效的原因：它将多重背包中数量为 s 的物品的枚举次数，从 O(s) 降到了 O(log s)，同时没有遗漏任何可能的最优解状态。(很大的恶问题:怎么证明没有遗漏任何解?)


# 代码

## 转换成01背包

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 10010;

int n,      // n为物品种类的数量  
    m,      // m是背包的整体容量上限  
    dp[N], // dp数组：用于状态转移以获得最大背包价值总和
    v[N],   // v数组存放每种物品的重量（或称体积占用）
    w[N],   // w数组存放每种物品对应的价值增益  
    k = 0; // k记录了将多种数量物品拆分后的新“物品总数”（即拆分后全部的01背包物品条目）

int main(){

    /* 接下指定输入：
        第一行的整数 n,m 表示了之后的n行所提供的物品详细信息：
       每行从左至右的三个整数:t_v,s,w 对应于：
       
       Example:
         
            4(bag weight capacity)     
            接下来的4行定义了多物品:
             1.weight 各自value amount数量   （通过 转换为 可行 数字内本问题小规模可直接单个重复添加进数组） 
             
       */
         
    /* 样例输入中输入：
          
            n=种类数量, m背包容量           
            接下来 n 行记录了：
                t_v:体积 
                t_w:价值  
                s: 最大可用数量 
                 
    */
    cin >> n >> m;  
    
    // 把多重背包问题转换为“01”背包
    for (int i=1; i <= n; i++) {
        int t_v, t_w, s;  // 当前 i类物品
        // 以当前该类物品 占体积 t_v，价值t_w, 和 s(最大可提供数量)  
        // 以1~s 逐一展开成为新 元素的数组
        cin >> t_v >> t_w >> s;  
        for (int j=1; j <= s; j++ ){ // j=1:最大s个,每下一个j 时
            v[++k] = t_v;  // 记录一个新的(唯一的)物品的体积
            w[k] = t_w;    // 及其价值
        }
    }
    
    // 经过上一段循环后, k为“总共”独特物品数量（每个物品只能使用一次）
    // 状态转移过程（ 01背包 ）  
    for (int i=1; i <= k; i++ ){  // 枚举每个(拆分好的)的单个物品i
        for (int j=m; j >= v[i]; j-- )  // 逆序更新,确保物品最多用一次(避免在多重背包的同一“原始物品”的多次拆分被反复使用) 
            // 状态更新：比较不选i和选i哪种更优,如果选择i, 则
            // dp[j] = 物品体积为j时的能装入的最大价值.
            dp[j] = max( dp[j], dp[j - v[i]] + w[i] );
    }
    
    // 输出最终的最高可能收益值：设能将任何子集的物品体积控制在背包总容量m以下  
    cout << dp[m] << endl;
}
```

## 二进制优化

```c++
#include<bits/stdc++.h>
using namespace std;
const int N=15000;  // 最大物品数：N*logS，S最大2000，log2000≈11，N=1000，所以需要约11000
int n,m,dp[N],v[N],w[N],cnt;  // cnt记录拆分后物品的总数
 
int main(){
    cin>>n>>m;  // 输入物品种数n和背包容量m
    
    for(int i=0;i<n;i++){
        int a,b,c;  // a:单个物品体积，b:单个物品价值，c:物品数量
        int k=1;    // 二进制拆分的基数，从1开始（2^0）
        cin>>a>>b>>c;
        
        // 二进制拆分核心：将c个物品拆分为1,2,4,...,2^k,c'的形式
        // 这样可以用log(c)个"物品包"表示0~c的所有选择[1](@ref)[4](@ref)
        while(k<=c){
            v[++cnt]=k*a;  // 第cnt个"物品包"的体积 = k个原物品的体积
            w[cnt]=k*b;    // 第cnt个"物品包"的价值 = k个原物品的价值
            c-=k;          // 减去已拆分的数量
            k=k*2;         // 基数翻倍，准备拆分下一组
        }
        
        // 处理剩余部分（如果c>0）
        if(c>0){
            v[++cnt]=c*a;  // 剩余c个物品作为一个"物品包"
            w[cnt]=c*b;
        }
        // 数学原理：任何0~s的整数都可以用1,2,4,...,2^k,c'这些数的子集和表示[4](@ref)[6](@ref)
        // 例如s=10，拆分为1,2,4,3，可以表示0~10的所有数[4](@ref)
    }
    
    // 经过二进制拆分后，问题转化为标准的01背包问题
    // 现在有cnt个"物品包"，每个包要么选要么不选[1](@ref)[2](@ref)
    for(int i=1;i<=cnt;i++){
        for(int j=m;j>=v[i];j--)  // 01背包逆序枚举，确保每个物品只选一次
            dp[j]=max(dp[j],dp[j-v[i]]+w[i]);  // 状态转移方程
    }
    
    cout<<dp[m]<<endl;  // 输出背包容量为m时的最大价值
    return 0;
}
```
