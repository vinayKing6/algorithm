思路:

先将所有人的装水桶时间按生序排序,按直觉讲第i个人需等待第1~i-1个人的打水时间,也就是说第i个人的打水时间被等待了n-i次(因为自己不用等待自己),所以最后结果就是 $\mathrm{res} = \sum t_i\times (n-i)$  

代码:

```c++
#include<bits/stdc++.h>  // 包含所有常用头文件（为竞赛方便）
using namespace std;
﻿

const int N = 100010;  // 定义最大数组长度
long n, a[N];  // n：人数， a[N]：存储每个人打水时间
﻿

int main() {
    cin >> n;  // 输入总人数
    
    // 输入每个人的打水时间
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    // 关键步骤：按打水时间升序排序
    // 贪心策略：让打水时间短的人先打，使总等待时间最小
    sort(a, a + n);
    
    long long res = 0;  // res：所有人等待时间的总和（用long long防止溢出）
    
    // 计算总等待时间
    // 当第i个人打水时，后面还有(n-i-1)个人需要等待他的打水时间
    // 第i个人的打水时间a[i]会成为后面所有人的共同等待时间
    for (int i = 0; i < n - 1; i++) {
        res += a[i] * (n - i - 1);
    }
    
    cout << res << endl;  // 输出总等待时间
    
    return 0;
}
﻿

/* 执行示例（对应文档中的样例）：
输入：
7
3 6 1 4 2 5 7
﻿

排序后数组：[1, 2, 3, 4, 5, 6, 7]
﻿

计算过程：
i=0: 1 * (7-0-1) = 1 * 6 = 6
i=1: 2 * (7-1-1) = 2 * 5 = 10
i=2: 3 * (7-2-1) = 3 * 4 = 12
i=3: 4 * (7-3-1) = 4 * 3 = 12
i=4: 5 * (7-4-1) = 5 * 2 = 10
i=5: 6 * (7-5-1) = 6 * 1 = 6
总等待时间 = 6 + 10 + 12 + 12 + 10 + 6 = 56
﻿

输出：56
*/
```
