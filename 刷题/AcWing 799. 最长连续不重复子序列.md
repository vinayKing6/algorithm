# 题目

https://www.acwing.com/problem/content/801/

给定一个长度为 n 的整数序列，请找出最长的不包含重复的数的连续区间，输出它的长度。

#### 输入格式

第一行包含整数 n。

第二行包含 n 个整数（均在 0∼$10^5$ 范围内），表示整数序列。

#### 输出格式

共一行，包含一个整数，表示最长的不包含重复的数的连续区间的长度。

#### 数据范围

1≤n≤$10^5$

#### 输入样例：

```
5
1 2 2 3 5
```

#### 输出样例：

```
3
```

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1e5 + 10;
int n, a[N], st[N], ans; // st数组充当哈希表，记录当前窗口内每个数字出现的次数

int main(){
    // 提高输入输出效率（ZJU机试推荐加上，防止大数据量超时）
    ios::sync_with_stdio(false); cin.tie(0);

    if(!(cin >> n)) return 0;
    for(int i = 1; i <= n; i++) cin >> a[i];

    // 双指针维护滑动窗口 [l, r]
    int l = 1, r = 1; 
    while(r <= n){
        // 1. 将右边界指向的数字加入窗口
        st[a[r]]++;

        // 2. 检查冲突：如果 a[r] 出现次数超过 1，说明窗口内有重复
        // 此时不断左移左指针 l，并从计数器中减去移出的数字，直到重复消失
        while(l < r && st[a[r]] > 1) {
            st[a[l]]--; // 移出左边界数字
            l++;        // 收缩窗口
        }

        // 3. 此时窗口 [l, r] 内一定没有重复数字，更新最大长度
        // 窗口长度计算公式：右下标 - 左下标 + 1
        ans = max(ans, r - l + 1);

        // 4. 右指针继续向右探索
        r++;
    }

    cout << ans << endl;
    return 0;
}
```
