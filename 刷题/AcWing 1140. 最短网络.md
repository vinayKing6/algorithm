# 题目

https://www.acwing.com/problem/content/1142/

农夫约翰被选为他们镇的镇长！

他其中一个竞选承诺就是在镇上建立起互联网，并连接到所有的农场。

约翰已经给他的农场安排了一条高速的网络线路，他想把这条线路共享给其他农场。

约翰的农场的编号是1，其他农场的编号是 2∼n。

为了使花费最少，他希望用于连接所有的农场的光纤总长度尽可能短。

你将得到一份各农场之间连接距离的列表，你必须找出能连接所有农场并使所用光纤最短的方案。

#### 输入格式

第一行包含一个整数 n，表示农场个数。

接下来 n 行，每行包含 n 个整数，输入一个对角线上全是0的对称矩阵。  
其中第 x+1 行 y 列的整数表示连接农场 x 和农场 y 所需要的光纤长度。

#### 输出格式

输出一个整数，表示所需的最小光纤长度。

#### 数据范围

3≤n≤100  
每两个农场间的距离均是非负整数且不超过100000。

#### 输入样例：

```
4
0  4  9  21
4  0  8  17
9  8  0  16
21 17 16  0
```

#### 输出样例：

```
28
```

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 110;   // 最大节点数
int n;               // 节点数
int g[N][N];         // 邻接矩阵，存储图的权值
int st[N];           // 标记节点是否已经加入最小生成树
int dist[N];         // dist[i] 表示节点 i 到当前生成树的最小边权值
int res;             // 最小生成树的总权值

// Prim算法求最小生成树
void prim(){
    memset(dist, 0x3f, sizeof dist); // 初始化dist为大数，表示无限远
    dist[1] = 0;                     // 从节点1开始，初始距离为0

    for(int i = 0; i < n; i++){      // 循环n次，每次选一个新节点加入MST
        int t = -1;                  // t用于存储当前距离MST最近的节点
        for(int j = 1; j <= n; j++){ // 遍历所有节点
            if(!st[j] && (t == -1 || dist[j] < dist[t])) // 找到未加入MST且距离最小的节点
                t = j;
        }

        st[t] = 1;          // 将节点t加入最小生成树
        res += dist[t];     // 累加最小边权值到总权值

        // 更新其他未加入MST的节点到MST的最小距离
        for(int k = 1; k <= n; k++){
            if(!st[k] && dist[k] > g[t][k])
                dist[k] = g[t][k];
        }
    }

    cout << res << endl; // 输出最小生成树总权值
}

int main(){
    cin >> n; // 输入节点数
    for(int i = 1; i <= n; i++){
        for(int j = 1; j <= n; j++){
            cin >> g[i][j]; // 输入图的邻接矩阵
        }
    }

    prim(); // 调用Prim算法
}
```
