# 问题描述

农民约翰的 $N$﻿ 头奶牛（编号为 $1..N$﻿）计划逃跑并加入马戏团，为此它们决定练习表演杂技。

奶牛们不是非常有创意，只提出了一个杂技表演：叠罗汉，表演时，奶牛们站在彼此的身上，形成一个高高的垂直堆叠。

奶牛们正在试图找到自己在这个堆叠中应该所处的位置顺序。这 $N$﻿ 头奶牛中的每一头都有着自己的重量 $W_i$﻿ 以及自己的强壮程度 $S_i$﻿。

一头牛支撑不住的可能性取决于它头上所有牛的总重量（不包括它自己）减去它的身体强壮程度的值，现在称该数值为风险值，风险值越大，这只牛撑不住的可能性越高。

您的任务是确定奶牛的排序，使得所有奶牛的风险值中的最大值尽可能的小。

输入格式

第一行输入整数 $N$﻿，表示奶牛数量。接下来 $N$﻿ 行，每行输入两个整数，表示牛的重量和强壮程度，第 $i$﻿ 行表示第 $i$﻿ 头牛的重量 $W_i$﻿ 以及它的强壮程度 $S_i$﻿。

输出格式

输出一个整数，表示最大风险值的最小可能值。

数据范围

$1 \le N \le 50000$﻿,﻿
$1 \le W_i \le 10,000$﻿,﻿
$1 \le S_i \le 1,000,000,000$﻿﻿

输入样例：310 32 53 3
输出样例：2

# 思路

每头牛的危险值计算公式为：
$$ \text{危险值} = \text{前面所有牛的总重量} - \text{自身的强壮值} $$

我们的目标是：通过调整奶牛的排列顺序，使得**所有奶牛的危险值中的最大值**尽可能小。

直观来说：
- **自身重量 $w_i$ 越大**的牛，应尽量放在下面（减少它“前面牛的重量之和”）；
- **自身强壮值 $s_i$ 越大**的牛，应尽量放在下面（增大减数，从而减小危险值）。

综合以上两点，我们猜测：按照 **$w_i + s_i$ 的值从小到大排序**（即升序排列）可能是最优方案。

考虑相邻的两头牛 $i$ 和 $i+1$，假设它们前面所有牛的重量之和为 $W$。

| 牛 | 交换前的危险值 | 交换后的危险值 |
| :--- | :--- | :--- |
| $i$ | $W - s_i$ | $W + w_{i+1} - s_i$ |
| $i+1$ | $W + w_i - s_{i+1}$ | $W - s_{i+1}$ |

交换前后，**其他牛的危险值不会改变**（因为 $i$ 之前的牛不受影响，$i+1$ 之后的牛只感受到它们两个的总重量 $w_i + w_{i+1}$，而这个和不变）。因此，我们只需要比较交换前后，这两头牛的危险值的最大值。

为了使交换后不会变得更差（即最大危险值不增加），我们需要满足：

$$ \max(W - s_i,\ W + w_i - s_{i+1}) \ge \max(W - s_{i+1},\ W + w_{i+1} - s_i) $$

为了简化比较，我们可以从四个值中都减去 $W$，得到等价条件：

$$ \max(-s_i,\ w_i - s_{i+1}) \ge \max(-s_{i+1},\ w_{i+1} - s_i) $$

可以证明（分类讨论），当且仅当 $w_i + s_i \le w_{i+1} + s_{i+1}$ 时，上式成立。也就是说，如果 $w_i + s_i > w_{i+1} + s_{i+1}$，那么交换 $i$ 和 $i+1$ 的位置会使最大危险值变小（或不变）。

因此，**将所有牛按照 $w_i + s_i$ 的值升序排列**，即可得到最优的堆叠顺序。

算法步骤

1.  **读入数据**：读取奶牛数量 $N$，以及每头牛的重量 $w_i$ 和强壮值 $s_i$。
2.  **排序**：将所有奶牛按照 $w_i + s_i$ 的值**从小到大**排序。
3.  **计算并更新最大危险值**：
    - 初始化一个变量 `sum` 为 0，用于累积前面所有牛的重量之和。
    - 初始化一个变量 `ans` 为一个很小的数（例如负无穷），用于记录最大危险值。
    - 按排序后的顺序遍历每一头牛：
        - 计算当前牛的危险值：`sum - s_i`。
        - 用这个危险值更新 `ans`。
        - 将这头牛的重量 $w_i$ 加到 `sum` 中。
4.  **输出结果**：最终 `ans` 即为所有奶牛风险值中的最大值的最小可能值，将其输出。

- **时间复杂度**：排序需要 $O(N \log N)$，遍历计算需要 $O(N)$，总体为 $O(N \log N)$，对于 $N \le 50000$ 的数据范围完全可行。
- **空间复杂度**：主要是存储奶牛数据，为 $O(N)$。

# 代码

```c++
#include<bits/stdc++.h> // 引入所有标准库，包括iostream、vector、algorithm等，方便竞赛使用
using namespace std;     // 使用标准命名空间，这样可以直接使用cin, cout, sort等，无需加std::前缀

const int N=50010;       // 定义常量N，表示奶牛的最大数量，略大于50000以防止数组越界
typedef pair<int,int> PII; // 定义PII为pair<int,int>的别名，用于存储奶牛的相关信息
                           // 在这里，pair的first元素将存储 (w + s)，second元素存储 w

int n;                   // 存储奶牛的数量
PII cow[N];              // 定义一个PII类型的数组，用于存储每头奶牛的数据

int main(){
    // 优化输入输出，提高程序运行效率，尤其在处理大量数据时
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    cin>>n; // 读取奶牛的数量N
    
    // 循环读取每头奶牛的重量w和强壮程度s
    for(int i=0; i<n; i++){
        int w,s;
        cin>>w>>s;
        // 根据贪心策略，我们将奶牛按照 w + s 的值升序排序
        // 所以在pair中，first元素存储 w + s，second元素存储 w
        cow[i]={w+s, w}; 
    }
    
    // 对奶牛数组进行排序。
    // pair默认按照first元素升序排序，如果first元素相同，则按照second元素升序排序。
    // 这里我们只需要按照 (w + s) 的值升序排序即可，这正是pair的默认行为。
    sort(cow, cow+n); 
    
    int res = INT_MIN; // 初始化最大风险值为C++中int类型的最小值，确保任何风险值都能正确更新它
    int sum = 0;       // 初始化当前堆叠在下方所有奶牛的总重量为0

    // 遍历排序后的奶牛数组，计算每头奶牛的风险值并找出最大风险
    for(int i=0; i<n; i++){
        // 计算当前奶牛的风险值：
        // 风险值 = 前面所有牛的总重量 - 自身的强壮值
        // 在我们的pair中：
        // cow[i].first 存储的是 (w_i + s_i)
        // cow[i].second 存储的是 w_i
        // 所以，s_i = cow[i].first - cow[i].second
        // 因此，当前奶牛的风险值 = sum - s_i = sum - (cow[i].first - cow[i].second)
        res = max(res, sum - (cow[i].first - cow[i].second));
        
        // 将当前奶牛的重量 (cow[i].second) 加入到总重量中，
        // 供下一头（即它上面一头）奶牛计算风险值时使用
        sum += cow[i].second;
    }

    cout<<res<<endl; // 输出所有奶牛中最大风险值的最小可能值
    
    return 0; // 程序正常结束
}

```
