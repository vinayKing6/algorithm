# 题目

https://www.acwing.com/problem/content/1102/

农夫知道一头牛的位置，想要抓住它。

农夫和牛都位于数轴上，农夫起始位于点 N，牛位于点 K。

农夫有两种移动方式：

1. 从 X 移动到 X−1 或 X+1，每次移动花费一分钟
2. 从 X 移动到 2∗X，每次移动花费一分钟

假设牛没有意识到农夫的行动，站在原地不动。

农夫最少要花多少时间才能抓住牛？

#### 输入格式

共一行，包含两个整数N和K。

#### 输出格式

输出一个整数，表示抓到牛所花费的最少时间。

#### 数据范围

0≤N,K≤$10^5$

#### 输入样例：

```
5 17
```

#### 输出样例：

```
4
```

# 思路

考察 bfs 知识点。

- 从起点开始，用 bfs 爆搜走到其他所有点的移动次数。
    
- 需要将走到过的点记录下来，因为同一个点可以被多种方式走到，我们只保留步数最少的方法（也就是第一次走到的方法）。
    
- 如果搜到了终点，就停止 bfs, 输出结果。

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

// N 定义为 2e5+10 是因为 k 最大可能是 1e5，
// 但在搜索过程中 u*2 可能会超过 1e5，所以空间要开大一点防止越界
const int N = 2e05 + 10;

int n, k;
int dist[N]; // dist[i] 存储从起点 n 到位置 i 的最短分钟数（同时充当 st 判重数组）
queue<int> q;

void bfs() {
    // 初始化距离为 -1，表示所有点尚未被访问
    memset(dist, -1, sizeof dist);
    
    // 起点入队
    q.push(n);
    dist[n] = 0; // 起点到自己的距离是 0
    
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        
        // 【目标判断】如果当前位置就是牛的位置，输出答案并结束
        if (u == k) {
            cout << dist[u] << endl;
            return; // 找到即最短，直接退出
        }
        
        // 每一步有三种选择：后退、前进、瞬移
        
        // 1. 后退一步：u - 1
        // 只要不越界 (>=0) 且该位置没被走过，就入队
        if (u - 1 >= 0 && dist[u - 1] == -1) {
            dist[u - 1] = dist[u] + 1;
            q.push(u - 1);
        }
        
        // 2. 前进一步：u + 1
        // 只要不超过数组边界且没被走过
        if (u + 1 < N && dist[u + 1] == -1) {
            dist[u + 1] = dist[u] + 1;
            q.push(u + 1);
        }
        
        // 3. 瞬移：u * 2
        // 翻倍位置同样需要检查边界和是否走过
        if (u * 2 < N && dist[u * 2] == -1) {
            dist[u * 2] = dist[u] + 1;
            q.push(u * 2);
        }
    }
}

int main() {
    // 读入起点 n 和终点 k
    if (cin >> n >> k) {
        // 特判：如果起点就在终点后面，只能一步步往回退
        if (n >= k) {
            cout << n - k << endl;
        } else {
            bfs();
        }
    }
    return 0;
}
```
