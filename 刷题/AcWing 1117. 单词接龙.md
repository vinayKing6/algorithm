# 题目

https://www.acwing.com/problem/content/description/1119/

单词接龙是一个与我们经常玩的成语接龙相类似的游戏。

现在我们已知一组单词，且给定一个开头的字母，要求出以这个字母开头的最长的“龙”，每个单词最多被使用两次。

在两个单词相连时，其重合部分合为一部分，例如 beast 和 astonish ，如果接成一条龙则变为 beastonish。

我们可以任意选择重合部分的长度，但其长度必须大于等于1，且严格小于两个串的长度，例如 at 和 atide 间不能相连。

#### 输入格式

输入的第一行为一个单独的整数 n 表示单词数，以下 n 行每行有一个单词（只含有大写或小写字母，长度不超过20），输入的最后一行为一个单个字符，表示“龙”开头的字母。

你可以假定以此字母开头的“龙”一定存在。

#### 输出格式

只需输出以此字母开头的最长的“龙”的长度。

#### 数据范围

n≤20，  
单词随机生成。

#### 输入样例：

```
5
at
touch
cheat
choose
tact
a
```

#### 输出样例：

```
23
```

#### 提示

连成的“龙”为 atoucheatactactouchoose。

# 思路

### 算法分析

- 1、通过贪心的思想，两个字符串拼接，重合部分越短，拼接后的长度越长，因此，需要预处理出所有单词两两拼接重合的最小长度,`g[][]`表示`i`位置子符串与`j`位置子符串拼接重合的最小长度
    
- 2、从给定的字符开始，确定单词龙的第一个单词，向所有能够拼接到第一个单词后面的单词进行深度优先遍历
    
- 3、`dfs(dragon,last)`：`dragon`表示当前的单词龙,`last`表示上一个连接到字符串的单词

这道题是经典的搜索题——**“单词接龙”**。它的核心目标是：给定一组单词和一个起始字符，将单词首尾相连（必须有重合部分，但不能包含关系），求能拼成的最长“龙”的长度。

每个单词在一条龙中最多只能使用 **2 次**。

---

### 1. 全局变量与预处理

C++

```
const int N = 21;
int n;
string word[N]; // 存储输入的 n 个单词
int g[N][N];    // 预处理数组：g[i][j] 表示单词 i 后面接单词 j 时，重合部分的最小长度
int used[N];    // 标记数组：记录每个单词在当前递归路径中被使用了几次（最多 2 次）
int ans;        // 存储最终的最长龙长度
```

### 2. 预处理重合长度（关键逻辑）

在 `main` 函数中，这段嵌套循环预计算了所有单词对之间的连接信息：

C++

```
for (int i = 0; i < n; i ++ )
    for (int j = 0; j < n; j ++ )
    {
        string a = word[i], b = word[j];
        // 枚举重合部分的长度 k，从 1 开始（必须有重合）
        // k 的上限是两个单词长度的最小值 - 1（因为题目要求不能存在包含关系）
        for (int k = 1; k < min(a.size(), b.size()); k ++ )
            // a 的后缀 k 位 等于 b 的前缀 k 位
            if (a.substr(a.size() - k, k) == b.substr(0, k))
            {
                g[i][j] = k; // 记录最小重合长度
                break;       // 关键：题目要求重合部分越短，龙越长，所以找到第一个满足的 k 就跳出
            }
    }
```

### 3. 深度优先搜索 `dfs`

C++

```
// dragon: 当前拼成的龙（字符串）
// last: 上一个加入龙的单词下标
void dfs(string dragon, int last)
{
    // 每次递归都更新最大长度
    ans = max((int)dragon.size(), ans);

    used[last] ++ ; // 标记：当前单词使用次数 +1

    for (int i = 0; i < n; i ++ )
        // 如果 单词 last 和 单词 i 可以接上，且 单词 i 使用次数未达 2 次
        if (g[last][i] && used[i] < 2)
            // 递归：拼接时去掉重合部分 word[i].substr(g[last][i])
            dfs(dragon + word[i].substr(g[last][i]), i);

    used[last] -- ; // 回溯：恢复现场，以便其他路径使用
}
```

### 4. 启动搜索

C++

```
char start;
cin >> start; // 读入龙的起始字母

for (int i = 0; i < n; i ++ )
    // 如果单词 word[i] 的首字母符合要求，将其作为龙的开头
    if (word[i][0] == start)
        dfs(word[i], i);
```

---

### 💡 核心考点总结

1. **重合规则**：
    
    - 必须有重合部分（`k >= 1`）。
        
    - 不能存在包含关系（例如 `at` 和 `atide` 不能接龙，因为 `at` 完全被包含）。
        
    - 为了使龙最长，重合部分应该**尽量短**（所以 `k` 从 1 开始枚举，搜到即 `break`）。
        
2. **回溯状态**：
    
    - `used[i] < 2` 是本题特有的约束。普通的 DFS 通常是 `used < 1`，这里允许每个单词用两次。
        
3. **预处理优化**：
    
    - 在 `dfs` 内部频繁进行字符串截取比较会非常慢。通过 `g[i][j]` 预处理，将 `dfs` 变成了纯粹的数值比较和路径查找。
        

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 22;
string w[N];        // 存储输入的 n 个单词
int n, g[N][N], ans, st[N]; 
// g[i][j] 记录单词 i 后面接单词 j 时，重合部分的最小长度
// st[i] 记录每个单词的使用次数（题目要求最多用 2 次）

/**
 * DFS 搜索函数
 * @param dragon: 当前已经拼成的龙字符串
 * @param last: 最后一个被加入龙中的单词下标
 */
void dfs(string dragon, int last){
    // 每次拼接出更长的龙，就更新最大长度 ans
    ans = max((int)dragon.size(), ans);
    
    // 尝试在当前的“龙”后面接上所有的单词 i
    for(int i = 0; i < n; i++){
        // 1. 如果单词 last 和单词 i 存在重合部分 (g[last][i] > 0)
        // 2. 且单词 i 在当前路径中被使用的次数小于 2 次
        if(g[last][i] && st[i] < 2){
            st[i]++; // 标记单词 i 使用次数 +1
            
            // 递归拼接：当前龙 + (单词 i 扣掉重合前缀后的剩余部分)
            // w[i].substr(k) 表示从下标 k 开始截取到末尾
            dfs(dragon + w[i].substr(g[last][i]), i);
            
            st[i]--; // 回溯：撤销对单词 i 的使用标记
        }
    }
}

int main(){
    cin >> n;
    for(int i = 0; i < n; i++) cin >> w[i];
    
    // --- 1. 预处理：计算任意两个单词之间的重合关系 ---
    for(int i = 0; i < n; i++){
        for(int j = 0; j < n; j++){
            string a = w[i], b = w[j];
            // 枚举可能的重合长度 k
            // 题目要求不能包含，所以 k 的上限是 min_len - 1
            for(int k = 1; k < min(a.size(), b.size()); k++){
                // 如果 a 的后 k 个字符 等于 b 的前 k 个字符
                if(a.substr(a.size() - k) == b.substr(0, k)){
                    g[i][j] = k; // 记录最小重合长度（重合越短，接出来的龙越长）
                    break;       // 找到最小的 k 就跳出，贪心思想
                }
            }
        }
    }
    
    // --- 2. 启动搜索 ---
    char start;
    cin >> start; // 读入龙的起始字母
    for(int i = 0; i < n; i++){
        // 只有首字母匹配的单词才能作为“龙头”
        if(w[i][0] == start){
            st[i]++;       // 标记龙头单词已使用 1 次
            dfs(w[i], i);  // 开始接龙
            st[i]--;       // 回溯（如果存在多个可能的龙头，清空标记供下次搜索）
        }
    }
    
    cout << ans << endl; // 输出最终的最大长度
}
```
