# 题目

https://www.acwing.com/problem/content/1133/

1944 年，特种兵麦克接到国防部的命令，要求立即赶赴太平洋上的一个孤岛，营救被敌军俘虏的大兵瑞恩。

瑞恩被关押在一个迷宫里，迷宫地形复杂，但幸好麦克得到了迷宫的地形图。

迷宫的外形是一个长方形，其南北方向被划分为 N 行，东西方向被划分为 M 列， 于是整个迷宫被划分为 N×M 个单元。

每一个单元的位置可用一个有序数对 (单元的行号, 单元的列号) 来表示。

南北或东西方向相邻的 2 个单元之间可能互通，也可能有一扇锁着的门，或者是一堵不可逾越的墙。

**注意：** 门可以从两个方向穿过，即可以看成一条无向边。

迷宫中有一些单元存放着钥匙，同一个单元可能存放 **多把钥匙**，并且所有的门被分成 P 类，打开同一类的门的钥匙相同，不同类门的钥匙不同。

大兵瑞恩被关押在迷宫的东南角，即 (N,M) 单元里，并已经昏迷。

迷宫只有一个入口，在西北角。

也就是说，麦克可以直接进入 (1,1) 单元。

另外，麦克从一个单元移动到另一个相邻单元的时间为 1，拿取所在单元的钥匙的时间以及用钥匙开门的时间可忽略不计。

试设计一个算法，帮助麦克以最快的方式到达瑞恩所在单元，营救大兵瑞恩。

#### 输入格式

第一行有三个整数,分别表示 N,M,P 的值。

第二行是一个整数 k，表示迷宫中门和墙的总数。

接下来 k 行，每行包含五个整数，$X_{i1},Y_{i1},X_{i2},Y_{i2},G_i$：当 $G_i≥1$ 时，表示 $(X_{i1},Y_{i1})$ 单元与 $(X_{i2},Y_{i2})$ 单元之间有一扇第 $G_i$ 类的门，当 $G_i=0$ 时，表示 $(X_{i1},Y_{i1})$ 单元与 $(X_{i2},Y_{i2})$ 单元之间有一面不可逾越的墙。

接下来一行，包含一个整数 S，表示迷宫中存放的钥匙的总数。

接下来 S 行，每行包含三个整数 $X_{i1},Y_{i1},Q_i$，表示 $(X_{i1},Y_{i1})$ 单元里存在一个能开启第 $Q_i$ 类门的钥匙。

#### 输出格式

输出麦克营救到大兵瑞恩的最短时间。

如果问题无解，则输出 -1。

#### 数据范围

|Xi1−Xi2|+|Yi1−Yi2|=1  
0≤Gi≤P  
1≤Qi≤P  
1≤N,M,P≤10  
1≤S≤14  
1≤k≤150

# 思路

思路有点类似:[[AcWing 1098. 城堡问题]]
0-1BFS也复习一下: [[AcWing 340. 通信线路]]

## 问题概述

这是一个 **迷宫 + 钥匙 + 门 + 最短路** 的问题。

目标：

- 从 `(1,1)` 走到 `(n,m)`
    
- 某些边是 **门**，需要对应 **钥匙**
    
- 某些格子有 **钥匙**
    
- 求 **最短步数**
    

核心难点：

- 同一个位置
    
- **是否拥有钥匙不同 → 状态不同**
    

因此状态必须设计为：

```
(位置, 钥匙集合)
```

---

## 状态设计

### 位置状态

二维网格：

```
(i, j)
```

转为一维编号：

```
g[i][j]
```

例如：

```
3×3

1 2 3
4 5 6
7 8 9
```

代码：

```cpp
g[i][j] = t++;
```

---

### 钥匙状态（状态压缩）

最多：

```
p ≤ 10
```

使用 **二进制表示钥匙集合**：

```
0000000000
```

表示没有钥匙

```
0000000101
```

表示拥有：

- 钥匙1
    
- 钥匙3
    

代码：

```cpp
state = t.y | key[t.x];
```

---

## 图的构建

### 邻接表结构

```cpp
int h[N*N];   // 每个节点的第一条边
int e[M];     // 边终点
int ne[M];    // 下一条边
int w[M];     // 门的钥匙编号
```

含义：

- `w = 0` → 普通路
    
- `w = c` → 需要钥匙 `c`
    

---

### 普通边构建

函数：

```cpp
build()
```

枚举每个格子四个方向：

```
上 右 下 左
```

如果合法：

```
add(a, b, 0)
```

表示 **普通移动**。

---

### 门的边

输入：

```
x1 y1 x2 y2 c
```

表示：

```
(x1,y1) ↔ (x2,y2)
需要钥匙 c
```

代码：

```cpp
add(a,b,c);
add(b,a,c);
```

---

## 钥匙存储

数组：

```
key[N*N]
```

含义：

```
key[i] = 该格子拥有的钥匙集合
```

代码：

```cpp
key[g[x][y]] |= 1 << (c-1);
```

例如：

```
key[5] = 0101
```

表示：

- 钥匙1
    
- 钥匙3
    

---

## BFS 状态设计

距离数组：

```
dist[位置][钥匙状态]
```

含义：

```
到达 (位置,钥匙集合) 的最短距离
```

访问数组：

```
st[位置][钥匙状态]
```

防止重复搜索。

队列存储：

```
(位置, 钥匙状态)
```

代码：

```cpp
deque<PII> q;
```

---

## BFS 初始化

起点：

```
位置 = 1
钥匙 = 0
```

代码：

```cpp
dist[1][0] = 0;
q.push_back({1,0});
```

---

## 状态转移

BFS 中有 **两种转移**。

---

### 1 移动到相邻格子

代码：

```cpp
dist[j][t.y] = dist[t.x][t.y] + 1;
q.push_back({j, t.y});
```

含义：

```
移动一步
代价 = 1
```

---

### 2 在当前位置拿钥匙

代码：

```cpp
int state = t.y | key[t.x];
dist[t.x][state] = dist[t.x][t.y];
q.push_front({t.x, state});
```

含义：

```
位置不变
钥匙增加
代价 = 0
```

---

## 为什么拿钥匙要插入队头

因为：

```
拿钥匙 = 0代价操作
```

移动：

```
代价 = 1
```

为了保证 **最短路顺序**：

|操作|队列位置|
|---|---|
|代价0|push_front|
|代价1|push_back|

这就是 **0-1 BFS**。

这样可以保证：

```
距离小的状态优先处理
```

我觉得因为钥匙少的能到下一个格子,钥匙多的也一定能到,且距离相等,但是钥匙多能到的地方更多,所以要插入队头.

---

## 为什么不能直接修改 state

错误写法：

```cpp
t.y |= key[t.x];
```

原因：

```
(位置,钥匙状态)
是图中的一个节点
```

例如：

```
(A ,000)
(A ,001)
```

是 **两个不同节点**。

拿钥匙实际上是：

```
(A ,000) → (A ,001)
```

一条 **权重为0的边**。

如果直接修改：

```
(A ,000) 会消失
```

会破坏 BFS 的正确性。

---

## 终点判断

当到达：

```
n * m
```

说明已经到达 `(n,m)`。

代码：

```cpp
if(t.x == n*m)
    return dist[t.x][t.y];
```

---

## 时间复杂度

位置数量：

```
n*m ≤ 100
```

钥匙状态：

```
2^10 = 1024
```

总状态：

```
100 × 1024 ≈ 10^5
```

每个状态最多扩展：

```
4 条边
```

复杂度：

```
O(n*m*2^p)
```

完全可行。

---

## 核心思想总结

这题本质是：

### 状态扩展最短路

把原图扩展为：

```
节点 = (位置,钥匙集合)
```

图规模：

```
(n*m) × (2^p)
```

边类型：

```
移动      权重1
拿钥匙    权重0
```

使用：

```
0-1 BFS
```

求最短路。

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

// N 最大网格边长
// P 钥匙状态数（最多10把钥匙 → 2^10种状态）
// M 最大边数
const int N=11,P=(1<<10),M=610;

typedef pair<int,int> PII;

// n,m 网格大小
// p 钥匙种类数
// k 门的数量
// s 钥匙数量
int n,m,p,k,s;

// 邻接表存图
int h[N*N],e[M],ne[M],idx,w[M];

// key[x] 表示节点x拥有的钥匙集合（bitmask）
int key[M];

// g[i][j] 将二维坐标映射为一维节点编号
int g[N][N];

// st[x][state] 表示状态是否已经访问
bool st[N*N][P];

// dist[x][state] 到达 (x,state) 的最短距离
int dist[N*N][P];

// edges 用于记录特殊边（带门的边）
set<PII> edges;

// 0-1 BFS 使用双端队列
deque<PII> q;

// 四个方向
int dx[]={-1,0,1,0};
int dy[]={0,1,0,-1};


// 添加一条边
// a → b
// c = 0 表示普通边
// c > 0 表示需要钥匙c
void add(int a,int b,int c){
    e[idx]=b;
    ne[idx]=h[a];
    w[idx]=c;
    h[a]=idx++;
}


// 构建普通移动边
// 如果两个格子之间没有门，则添加权重为0的普通边
void build(){
    for(int i=1;i<=n;i++){
        for(int j=1;j<=m;j++){
            for(int u=0;u<4;u++){
                int x=dx[u]+i;
                int y=dy[u]+j;

                // 越界判断
                if(x<=0 || x>n || y<=0 || y>m) continue;

                // 如果这条边已经是门，就不添加普通边
                if(edges.count({g[i][j],g[x][y]})) continue;

                add(g[i][j],g[x][y],0);
            }
        }
    }
}


// 0-1 BFS 求最短路
// 状态：(位置,钥匙集合)
int bfs(){

    memset(dist,0x3f,sizeof dist);

    // 起点：节点1，没有钥匙
    dist[1][0]=0;
    q.push_back({1,0});
    
    while(!q.empty()){
        PII u=q.front();
        int x=u.first;
        int state=u.second;
        q.pop_front();
        
        if(st[x][state]) continue;
        st[x][state]=1;
        
        // 到达终点
        if(x == n*m)
            return dist[x][state];
        
        // 当前格子如果有钥匙，可以更新钥匙状态
        if(key[x]){
            int newState = state | key[x];

            // 拿钥匙不增加距离（权重0）
            if(dist[x][newState]>dist[x][state]){
                dist[x][newState]=dist[x][state];

                // 0权重 → 插入队头
                q.push_front({x,newState});
            }
        }
        
        // 枚举所有邻边
        for(int i=h[x];~i;i=ne[i]){
            int j=e[i];

            if(st[j][state]) continue;

            // 如果是门，但没有对应钥匙
            if(w[i] && !((state>>(w[i]-1))&1)) continue;
            
            // 普通移动，代价为1
            if(dist[j][state] > dist[x][state]+1){
                dist[j][state]=dist[x][state]+1;

                // 权重1 → 插入队尾
                q.push_back({j,state});
            }
        }
    }
    
    // 无法到达终点
    return -1;
}

int main(){

    cin>>n>>m>>p>>k;

    memset(h,-1,sizeof h);
    
    // 将二维坐标映射为一维编号
    for(int i=1,t=0;i<=n;i++){
        for(int j=1;j<=m;j++){
            g[i][j]=++t;
        }
    }
    
    // 读入门的信息
    while(k--){
        int a,b,c,d,e;
        cin>>a>>b>>c>>d>>e;

        // 记录这条边为特殊边
        edges.insert({g[a][b],g[c][d]});
        edges.insert({g[c][d],g[a][b]});
        
        // 如果有门，则添加带权边
        if(e){
            add(g[a][b],g[c][d],e);
            add(g[c][d],g[a][b],e);
        }
    }

    // 读入钥匙
    cin>>s;
    while(s--){
        int a,b,c;
        cin>>a>>b>>c;

        // 记录钥匙
        key[g[a][b]] |= 1<<(c-1);
    }

    // 构建普通边
    build();

    // 输出最短路径
    cout<<bfs()<<endl;
}
```
