# 题目

https://www.acwing.com/problem/content/899/

给定两个长度分别为 N 和 M 的字符串 A 和 B，求既是 A 的子序列又是 B 的子序列的字符串长度最长是多少。

输入格式

第一行包含两个整数 N 和 M。  
第二行包含一个长度为 N 的字符串，表示字符串 A。  
第三行包含一个长度为 M 的字符串，表示字符串 B。  
字符串均由小写字母构成。

输出格式

输出一个整数，表示最大长度。

数据范围

1 ≤ N, M ≤ 1000

输入样例

```
4 5  
acbd  
abedc
```

输出样例

```
3
```

# 代码

```c++
/*
 * 题目：897. 最长公共子序列（Longest Common Subsequence, LCS）
 * 核心算法：动态规划
 * 时间/空间限制：1s / 64MB
 */

#include<bits/stdc++.h>  // 万能头文件，包含常见的STL库
using namespace std;

const int N = 1010;  // 数组最大长度，预留足够空间

char a[N], b[N];  // 存储两个输入字符串，下标从1开始
int f[N][N];     // DP状态数组，f[i][j]表示a前i个字符和b前j个字符的LCS长度
int n, m;        // 分别表示字符串a和b的长度

int main() {
    // 输入：n m 字符串a 字符串b
    // 注意：a+1表示从a[1]开始存储，使得下标从1开始，避免边界处理问题
    cin >> n >> m >> a + 1 >> b + 1;
    
    // 动态规划求解LCS长度
    for (int i = 1; i <= n; i++) {            // 遍历字符串a的每个字符
        for (int j = 1; j <= m; j++) {        // 遍历字符串b的每个字符
            // 状态转移方程：
            // 情况1：不包含a[i]的LCS长度（来自f[i-1][j]）
            // 情况2：不包含b[j]的LCS长度（来自f[i][j-1]）
            f[i][j] = max(f[i - 1][j], f[i][j - 1]);
            
            // 情况3：如果a[i]等于b[j]，则可以选择同时包含它们
            // 此时LCS长度 = f[i-1][j-1] + 1
            if (a[i] == b[j]) {
                f[i][j] = max(f[i][j], f[i - 1][j - 1] + 1);
            }
        }
    }
    
    // 输出最终结果：f[n][m]即为两个完整字符串的最长公共子序列长度
    cout << f[n][m] << endl;
    
    return 0;
}

/* 算法说明：
 * 1. 状态定义：f[i][j] 表示字符串a的前i个字符和字符串b的前j个字符的最长公共子序列长度
 * 2. 状态转移：
 *    - 若 a[i] == b[j]，则 f[i][j] = max(f[i-1][j], f[i][j-1], f[i-1][j-1] + 1)
 *    - 若 a[i] != b[j]，则 f[i][j] = max(f[i-1][j], f[i][j-1])
 * 3. 初始条件：f[j] = f[i][0] = 0（空字符串与任何字符串的LCS长度为0）
 * 4. 时间复杂度：O(n * m)
 * 5. 空间复杂度：O(n * m)
 */
```
