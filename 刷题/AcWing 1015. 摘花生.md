# 题目

Hello Kitty想摘点花生送给她喜欢的米老鼠。

她来到一片有网格状道路的矩形花生地(如下图)，从西北角进去，东南角出来。

地里每个道路的交叉点上都有种着一株花生苗，上面有若干颗花生，经过一株花生苗就能摘走该它上面所有的花生。

Hello Kitty只能向东或向南走，不能向西或向北走。

问Hello Kitty最多能够摘到多少颗花生。

![[Pasted image 20260305145930.png]]

#### 输入格式

第一行是一个整数T，代表一共有多少组数据。

接下来是T组数据。

每组数据的第一行是两个整数，分别代表花生苗的行数R和列数 C。

每组数据的接下来R行数据，从北向南依次描述每行花生苗的情况。每行数据有C个整数，按从西向东的顺序描述了该行每株花生苗上的花生数目M。

#### 输出格式

对每组输入数据，输出一行，内容为Hello Kitty能摘到得最多的花生颗数。

#### 数据范围

1≤T≤100,  
1≤R,C≤100,  
0≤M≤1000

#### 输入样例：

```
2
2 2
1 1
3 4
2 3
2 3 4
1 6 5
```

#### 输出样例：

```
8
16
```

# 思路

复习: [[AcWing 898. 数字三角形]]


1. 状态表示  
    集合：定义`f[i][j]`为从(1, 1)到达(i, j)的所有方案  
    属性：最大值
2. 状态转移  
    (i, j)从(i-1, j)即上方过来  
    (i, j)从(i, j-1)即左方过来
3. 滚动数组(可选) 
    `f[i][j]`只需要用到这一层和上一层的f元素，所以可以压缩成滚动数组。在此之上，还可以直接压缩成一维数组。

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 110;
int q, n, m;
int f[N][N]; // 状态数组：f[i][j] 表示从 (1,1) 走到 (i,j) 的所有路径中，获得的最大价值之和
int t[N][N]; // 地图数组：存储每个格子里的数字（价值）

int main() {
    // 读入测试数据组数 q
    cin >> q;
    while (q--) {
        // 读入矩阵的行数 n 和列数 m
        cin >> n >> m;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                cin >> t[i][j]; // 读入每个格子的数值
            }
        }
        
        // 初始化：每一组数据开始前清空状态数组
        memset(f, 0, sizeof f);
        
        // 核心动态规划过程
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                // 状态转移方程：
                // 要到达格子 (i,j)，只有两种方式：
                // 1. 从上方格子 (i-1, j) 向下走一步
                // 2. 从左方格子 (i, j-1) 向右走一步
                // 我们取这两者中的最大值，再加上当前格子本身的价值 t[i][j]
                f[i][j] = max(f[i - 1][j], f[i][j - 1]) + t[i][j];
            }
        }
        
        // 输出从左上角到右下角的最大价值总和
        cout << f[n][m] << endl;
    }
    return 0;
}
```
