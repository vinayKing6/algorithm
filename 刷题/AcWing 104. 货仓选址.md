题目描述

在一条数轴上有 $N$ 家商店，它们的坐标分别为 $A_1 \sim A_N$。

现在需要在数轴上建立一家货仓，每天清晨，从货仓到每家商店都要运送一车商品。

为了提高效率，求把货仓建在何处，可以使得货仓到每家商店的距离之和最小。

输入格式

第一行输入整数 $N$。

第二行 $N$ 个整数 $A_1 \sim A_N$。

输出格式

输出一个整数，表示距离之和的最小值。

数据范围

$1 \le N \le 100000$
$0 \le A_i \le 40000$

输入样例
```
4
6 2 9 1
```

输出样例：

```
12
```

思路:

货仓选址问题是一个经典的优化问题，其目标是在数轴上选择一个点作为货仓位置，使得该点到所有商店的距离之和最小。通过分析多个算法资源，我们可以系统地理解该问题的贪心解法及其数学证明。给定数轴上 $N$﻿ 个商店的坐标 $A_1, A_2, \dots, A_N$﻿，需要确定一个点 $x$﻿，使得目标函数：﻿$f(x) = \sum_{i=1}^N |A_i - x|$﻿﻿取得最小值。

核心结论：将商店坐标排序后，取中位数作为货仓位置 $x$﻿，即可使 $f(x)$﻿ 最小。具体而言：若 $N$﻿ 为奇数，则 $x = A_{\lfloor N/2 \rfloor}$﻿（排序后的中间值）。若 $N$﻿ 为偶数，则 $x$﻿ 可取 $A_{N/2 - 1}$﻿ 与 $A_{N/2}$﻿ 之间的任意位置（包括端点）。

为什么中位数是最优解将商店坐标按升序排列：$A_1 \le A_2 \le \dots \le A_N$﻿。考虑将距离配对分组：﻿$f(x) = (|A_1 - x| + |A_N - x|) + (|A_2 - x| + |A_{N-1} - x|) + \dots$﻿﻿对于任意一对 $(A_i, A_{N+1-i})$﻿，由绝对值不等式的几何意义可知：﻿$|A_i - x| + |A_{N+1-i} - x| \ge A_{N+1-i} - A_i$﻿﻿当且仅当 $x$﻿ 位于区间 $[A_i, A_{N+1-i}]$﻿ 内时取等号。因此，要使 $f(x)$﻿ 最小，$x$﻿ 必须同时位于所有这样的区间内，即位于最中间的一个或两个点之间，这正是中位数位置。算法步骤:排序：将数组 $A$﻿ 按升序排序。取中位数：计算下标 $mid = N / 2$﻿（整数除法）。计算距离和：遍历所有商店，累加 $|A_i - A_{mid}|$﻿。输出结果。时间复杂度：排序 $O(N \log N)$﻿，求和 $O(N)$﻿，总复杂度 $O(N \log N)$﻿。空间复杂度：$O(N)$﻿ 存储坐标，或 $O(1)$﻿ 若使用原地排序。

代码
```c++
#include<bits/stdc++.h>

using namespace std;
const int N=100010;  // 定义最大数组大小

int dist[N], n;      // dist数组存储商店位置，n表示商店数量

int main(){

​    // 输入商店数量

​    cin >> n;

​    // 输入每个商店的位置坐标

​    for(int i = 0; i < n; i++)

​        cin >> dist[i];

​    

​    // 对商店位置进行排序

​    // 排序后可以快速找到中位数位置

​    sort(dist, dist + n);

​    

​    int res = 0;  // 存储最终的最小距离和

​    

​    // 计算所有商店到中位数位置的距离之和

​    // dist[n/2] 是中位数位置（当n为偶数时取偏左中位数）

​    // 货仓建在中位数位置可以使总距离最小（数学证明）

​    for(int i = 0; i < n; i++)

​        res += abs(dist[i] - dist[n/2]);  // 累加绝对距离

​    

​    // 输出最小距离和

​    cout << res << endl;

​    

​    return 0;

}
```
