# 题目

[https://www.acwing.com/activity/content/problem/content/1002/](https://www.acwing.com/problem/content/900/)

给定一个如下图所示的数字三角形，从顶部出发，在每一结点可以选择移动至其左下方的结点或移动至其右下方的结点，一直走到底层，要求找出一条路径，使路径上的数字的和最大。

```
        7
      3   8
    8   1   0
  2   7   4   4
4   5   2   6   5
```

**输入格式**

第一行包含整数 n，表示数字三角形的层数。

接下来 n 行，每行包含若干整数，其中第 i 行表示数字三角形第 i 层包含的整数。

**输出格式**

输出一个整数，表示最大的路径数字和。

**数据范围**

1 ≤ n ≤ 500,  
-10000 ≤ 三角形中的整数 ≤ 10000

**输入样例**

```
5
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5
```

**输出样例**

```
30
```

# 思路

## 自底向上动态规划

**闫氏DP分析法：**
![[Pasted image 20260205200716.png]]
### 状态表示

- **集合**：从(i,j)到最后一层的所有路径方案
- **属性**：Max（最大值）

### 状态计算

- 点(i,j)可以走到左下方点(i+1,j)：`f[i+1][j] + a[i][j]`
- 点(i,j)也可以走到右下方点(i+1,j+1)：`f[i+1][j+1] + a[i][j]`
- **状态转移方程**：
    
    ```
    f[i][j] = max(f[i+1][j], f[i+1][j+1]) + a[i][j]
    ```
    

# 代码

```c++
#include<bits/stdc++.h> // 包含常用库
using namespace std;
const int N=510; // 定义最大层数为510

int n,dp[N][N],t[N][N]; // n:层数, dp:状态数组, t:存储输入的数字三角形

int main(){
    cin>>n; // 输入层数n
    
    // 读入数字三角形
    for(int i=1;i<=n;i++){
        for(int j=1;j<=i;j++){
            cin>>t[i][j]; // 存储第i行第j列的值
        }
    }
    
    // 初始化dp数组（从底向上计算）
    for(int j=1;j<=n;j++)
        dp[n][j]=t[n][j]; // 最后一行（底层）的初始值就是三角形底层的值
        
    // 从倒数第二行开始向上计算
    for(int i=n-1;i>=1;i--){
        for(int j=1;j<=i;j++)
            // 核心状态转移方程：
            // 当前(i,j)位置的最大值 = 当前值 + 
            // 下一行相邻两个位置（左下和右下）的最大值
            dp[i][j]=max(dp[i+1][j],dp[i+1][j+1])+t[i][j];
    }
    
    // 输出结果：dp[1]就是顶点到底层的最大路径和
    cout<<dp[1][1]<<endl;
    return 0;
}
```
