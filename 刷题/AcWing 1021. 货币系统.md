# 题目

https://www.acwing.com/problem/content/1023/

给你一个n种面值的货币系统，求组成面值为m的货币有多少种方案。

#### 输入格式

第一行，包含两个整数n和m。

接下来n行，每行包含一个整数，表示一种货币的面值。

#### 输出格式

共一行，包含一个整数，表示方案数。

#### 数据范围

n≤15,m≤3000

#### 输入样例：

```
3 10
1
2
5
```

#### 输出样例：

```
10
```

# 思路
参考[[AcWing 1023. 买书]],思路一模一样

# 代码

```c++
#include<bits/stdc++.h> // 包含所有标准库的头文件（万能头）
using namespace std;

// N 为货币种数上限，M 为目标金额上限
const int N=16, M=3010; 

int n, m, h[N]; // n: 货币种类, m: 目标金额, h[i]: 第 i 种货币的面值
long long f[M]; // f[j] 表示凑出金额为 j 的方案总数（结果可能很大，用 long long）

int main(){
    // 输入货币种类数和目标金额
    cin >> n >> m;
    
    // 输入每种货币的具体面额
    for(int i=1; i<=n; i++) cin >> h[i];
    
    // 初始化：凑出金额 0 的方案只有一种，即“什么都不选”
    f[0] = 1; 
    
    // 动态规划核心部分：完全背包求方案数
    for(int i=1; i<=n; i++){         // 遍历每一种货币
        for(int j=h[i]; j<=m; j++){  // 正向遍历金额（完全背包特征，可重复叠加）
            // 状态转移方程：
            // 当前凑出金额 j 的方案数 = 之前的方案数 + 消耗一个 h[i] 后剩余金额的方案数
            f[j] += f[j - h[i]];
        }
    }
    
    // 输出凑出目标金额 m 的最终方案总数
    cout << f[m] << endl;
    
    return 0;
}
```
