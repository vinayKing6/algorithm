# 题目

https://www.acwing.com/problem/content/862/

给定一个 n 个点 m 条边的无向图，图中可能存在重边和自环。

请你判断这个图是否是二分图。

#### 输入格式

第一行包含两个整数 n 和 m。

接下来 m 行，每行包含两个整数 u 和 v，表示点 u 和点 v 之间存在一条边。

#### 输出格式

如果给定图是二分图，则输出 `Yes`，否则输出 `No`。

#### 数据范围

1≤n,m≤105

#### 输入样例：

```
4 4
1 3
1 4
2 3
2 4
```

#### 输出样例：

```
Yes
```

# 思路

**什么叫二分图**

- 有两顶点集且图中每条边的的两个顶点分别位于两个顶点集中，每个顶点集中没有边直接相连接！
    
- 说人话的定义：图中点通过移动能分成左右两部分，左侧的点只和右侧的点相连，右侧的点只和左侧的点相连。

**如果判断一个图是不是二分图？**

- 开始对任意一未染色的顶点染色。
    
- 判断其相邻的顶点中，若未染色则将其染上和相邻顶点不同的颜色。
    
- 若已经染色且颜色和相邻顶点的颜色相同则说明不是二分图，若颜色不同则继续判断。
    
- bfs和dfs可以搞定！

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1e5 + 10; // 点的规模
const int M = 2e5 + 10; // 边权规模，因为是无向图，数组要开到边数的两倍

int n, m;
int h[N], e[M], ne[M], idx; // 邻接表存储图
int color[N]; // 存储每个点的颜色：0表示未染色，1表示颜色1，2表示颜色2

// 邻接表加边函数
void add(int a, int b){
    e[idx] = a;      // 存储边的终点
    ne[idx] = h[b];  // 将当前边指向点 b 的上一条边
    h[b] = idx++;    // 更新点 b 的头结点
}

// DFS 染色函数
// u: 当前待染色的点, c: 准备染上的颜色
bool dfs(int u, int c){
    color[u] = c; // 给当前点染色
    
    // 遍历当前点 u 的所有邻居节点
    for(int i = h[u]; i != -1; i = ne[i]){
        int v = e[i]; // v 是 u 的邻居
        
        if(!color[v]){ 
            // 如果邻居 v 还没有染色，则递归给它染上相反的颜色 (3 - c)
            // (如果 c=1, 则 3-1=2; 如果 c=2, 则 3-2=1)
            if(!dfs(v, 3 - c)) return false; 
        } 
        else if(color[v] == c){ 
            // 如果邻居 v 已经染过色，且颜色和当前点 u 的颜色 c 相同
            // 说明发生了冲突，图中存在奇环，不是二分图
            return false;
        }
    }
    
    return true; // 染色过程无冲突
}

int main(){
    // 读入点数 n 和边数 m
    if (!(cin >> n >> m)) return 0;
    
    memset(h, -1, sizeof h); // 初始化邻接表头结点为 -1
    
    while(m--){
        int a, b;
        cin >> a >> b;
        add(a, b); // 无向图加边：a -> b
        add(b, a); // 无向图加边：b -> a
    }
    
    // 判定二分图的关键：遍历所有点
    for(int i = 1; i <= n; i++){
        // 如果点 i 还没有被染色，说明它属于一个新的连通分量
        if(!color[i]){
            // 开始从点 i 进行 DFS 染色，赋予初始颜色 1
            if(!dfs(i, 1)){
                // 如果染色过程中发现冲突，直接输出 No 并结束程序
                cout << "No" << endl;
                return 0;
            }
        }
    }
    
    // 如果所有连通分量都能成功染色且无冲突，则是二分图
    cout << "Yes" << endl;
    
    return 0;
}
```
