# 题目

https://www.acwing.com/problem/content/1026/

有一个箱子容量为 V，同时有 n 个物品，每个物品有一个体积（正整数）。

要求 n 个物品中，任取若干个装入箱内，使箱子的剩余空间为最小。

#### 输入格式

第一行是一个整数 V，表示箱子容量。

第二行是一个整数 n，表示物品数。

接下来 n 行，每行一个正整数（不超过10000），分别表示这 n 个物品的各自体积。

#### 输出格式

一个整数，表示箱子剩余空间。

#### 数据范围

0<V≤20000,  
0<n≤30

#### 输入样例：

```
24
6
8
3
12
7
9
7
```

#### 输出样例：

```
0
```

### 算法1

##### (二维DP) O(n^2)

现在我们把这个问题确定为包装过的[01背包](https://www.acwing.com/file_system/file/content/whole/index/content/4476078/)（但是这道题没有[采药](https://www.acwing.com/file_system/file/content/whole/index/content/4538614/)这道题这么直白就是了）

那我们把物品i放入箱子里可以获得什么呢

i物品的体积。

看图：

![[Pasted image 20260305163316.png]]

注意：

> 现在把物品放入箱子中没有价值，而是体积  
> 在输入每个物品时，不要再多输入个价值  
> 输入是先箱子最大容量（V），再输入物品个数（N）  
> 特别坑的是，f 数组的第二位必须设成 M×30，（M为物品各自的体积，30 的含义是最多 30 个物品）


# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N=20010;  // 背包容量的最大值
int n,m;            // n：物品数量，m：背包容量
int v[31];          // 每个物品的重量，最多30个物品
int f[N];           // 动态规划数组 f[j] 表示容量为 j 的背包可以装的最大重量

int main(){
    cin>>m>>n;  // 输入背包容量 m 和物品数量 n

    // 输入每个物品的重量
    for(int i=1;i<=n;i++)
        cin>>v[i];

    // 01背包动态规划，注意倒序遍历 j
    for(int i=1;i<=n;i++){       // 遍历每个物品
        for(int j=m;j>=v[i];j--){  // 倒序遍历背包容量，防止重复使用同一个物品
            f[j]=max(f[j],f[j-v[i]]+v[i]);  // 状态转移：
            // f[j]：当前容量最大能装重量
            // f[j-v[i]] + v[i]：放入当前物品后的重量
            // 取两者较大值
        }
    }

    // 输出剩余容量
    cout<<m-f[m]<<endl;  // 背包剩余容量 = 总容量 - 可装最大重量
}
```
