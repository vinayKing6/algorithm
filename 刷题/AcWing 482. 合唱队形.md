# 题目

[482. 合唱队形 - AcWing题库](https://www.acwing.com/problem/content/description/484/)

N 位同学站成一排，音乐老师要请其中的 (N−K) 位同学出列，使得剩下的 K 位同学排成合唱队形。     

合唱队形是指这样的一种队形：设 K 位同学从左到右依次编号为 1，2…，K 他们的身高分别为 $T_1，T_2，…，T_K$，则他们的身高满足 $T_1<…<T_i>T_{i+1}>…>T_K$ (1≤i≤K)。

你的任务是，已知所有 N 位同学的身高，计算最少需要几位同学出列，可以使得剩下的同学排成合唱队形。

#### 输入格式

输入的第一行是一个整数 N，表示同学的总数。

第二行有 N 个整数，用空格分隔，第 i 个整数 $T_i$ 是第 i 位同学的身高(厘米)。

#### 输出格式

输出包括一行，这一行只包含一个整数，就是最少需要几位同学出列。

#### 数据范围

2≤N≤100
130≤Ti≤230

#### 输入样例：

```
8
186 186 150 200 160 130 197 220
```

#### 输出样例：

```
4
```

# 思路
参考[[AcWing 1014. 登山]]
一模一样，唯一区别是最后输出的是出队的人数

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1010;
int n, h[N], f1[N], f2[N];

int main(){
    cin >> n;
    for(int i = 1; i <= n; i++) cin >> h[i];

    // 1. 第一遍 DP：从左向右求最长上升子序列 (LIS)
    // f1[i] 表示以第 i 个元素结尾的最长上升子序列的长度
    for(int i = 1; i <= n; i++){
        f1[i] = 1; // 初始长度为 1（自身）
        for(int j = 1; j <= i - 1; j++){
            if(h[j] < h[i]) // 如果前面的比当前小，说明可以构成上升序列
                f1[i] = max(f1[i], f1[j] + 1);
        }
    }

    // 2. 第二遍 DP：从右向左求最长下降子序列 (LDS)
    // 或者理解为从右向左看的“最长上升子序列”
    // f2[i] 表示以第 i 个元素开头的最长下降子序列的长度
    for(int i = n; i >= 1; i--){
        f2[i] = 1; // 初始长度为 1
        for(int j = n; j >= i + 1; j--){
            if(h[j] < h[i]) // 如果后面的比当前小，说明可以构成下降序列
                f2[i] = max(f2[i], f2[j] + 1);
        }
    }

    // 3. 枚举中心点（山峰），求最大合唱队形长度
    int max_ = 0;
    for(int i = 1; i <= n; i++){
        /* 以 i 为顶点时，队形长度 = 左侧上升序列长度 + 右侧下降序列长度 - 1
           减 1 是因为第 i 个人在 f1[i] 和 f2[i] 中被重复计算了两次
        */
        max_ = max(max_, f1[i] + f2[i] - 1);
    }

    // 4. 题目要求的是“最少出列人数”
    // 最少出列人数 = 总人数 - 最长合唱队形长度
    cout << n - max_ << endl;

    return 0;
}
```
