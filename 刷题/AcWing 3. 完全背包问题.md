# 题目描述

https://www.acwing.com/problem/content/3/

问题描述

有 **N** 种物品和一个容量是 **V** 的背包，每种物品都有无限件可用。

第 **i** 种物品的体积是 **vᵢ**，价值是 **wᵢ**。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。输出最大价值。

输入格式

第一行两个整数，**N**，**V**，用空格隔开，分别表示物品种数和背包容积。

接下来有 **N** 行，每行两个整数 **vᵢ, wᵢ**，用空格隔开，分别表示第 **i** 种物品的体积和价值。

输出格式

输出一个整数，表示最大价值。

数据范围

0 < N, V ≤ 1000  
0 < vᵢ, wᵢ ≤ 1000

输入样例

```
4 5
1 2
2 4
3 4
4 5
```

输出样例

```
10
```

## **第一次优化（空间未优化）**

- 观察递推关系：
    - `f[i][j] = max(f[i-1][j], f[i-1][j-v]+w, f[i-1][j-2v]+2w, ...)`
    - `f[i][j-v] = max(f[i-1][j-v], f[i-1][j-2v]+w, f[i-1][j-3v]+2w, ...)`
- 发现：`f[i][j]` 的所有项（除第一项外）与 `f[i][j-v]` 的对应项只差一个 w
- 得到优化后的状态转移方程：
    
    ```
    f[i][j] = max(f[i-1][j], f[i][j-v[i]] + w[i])
    ```
    
- 含义：
    - **选择 0 件 i**：`f[i-1][j]`（继承前 i-1 种物品的结果）
    - **选择至少 1 件 i**：`f[i][j-v[i]] + w[i]`（在同种物品已经考虑的基础上再加一件）  
        ci-1**对比 01 背包**（核心区别）：
    - 01 背包（物品只能选 0/1 件）：`f[i][j] = max(f[i-1][j], f[i-1][j-v[i]] + w[i])`
    - 完全背包（物品可多选）：`f[i][j] = max(f[i-1][j], f[i][j-v[i]] + w[i])`
    - 区别：第二项的状态维度是 **i**（当前物品）而不是 **i-1**，因为可以重复选用当前物品

## **空间优化（一维数组）**

- 基于上述优化的状态转移方程，可以进一步优化空间：
    - 去掉第一维 i，只保留容量维度
    - 更新顺序：**必须正向枚举 j（从 v[i] 到 V）**
        - 因为 `f[j]` 的更新依赖于 **更新后** 的 `f[j-v[i]]`（即本层 i 的结果）
        - 这与 01 背包的逆向枚举正好相反
- 最终状态转移方程：
	```
	f[j] = max(f[j], f[j - v[i]] + w[i])，其中 j 从 v[i] 递增到v
	```
## **核心要点总结**

1. **物品无限**是本质特征，允许重复选择同一物品
2. **状态转移核心**：直接引用 **当前物品层** 的较小体积状态（`f[i][j-v[i]]`）
3. **空间优化关键**：正向遍历背包容量，保证复用本层更新结果
4. **与 01 背包对比**：
    - 状态转移方程：`f[i][j-v[i]]` vs `f[i-1][j-v[i]]`
    - 空间优化遍历方向：正向 vs 逆向

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;
const int N=1010;  // 定义常量 N，表示物品数量和背包容量上限

// n 是物品数量，m 是背包总容量
// w[N] 存储物品价值（weight/value）
// v[N] 存储物品体积（volume/weight）
// f[N] 是动态规划数组，f[j] 表示容量为 j 时能获得的最大价值
int n, m, w[N], v[N], f[N];

int main() {
    // 输入物品数量 n 和背包容量 m
    cin >> n >> m;
    
    // 读入 n 个物品的信息，v[i] 是体积，w[i] 是价值
    for(int i = 1; i <= n; i++)
        cin >> v[i] >> w[i];
    
    // 完全背包问题动态规划主循环
    for(int i = 1; i <= n; i++) {          // 遍历所有物品
        for(int j = v[i]; j <= m; j++) {   // 正序遍历容量（因为物品可以无限次使用）
            // 状态转移方程：考虑第 i 个物品时
            // f[j] 表示不选择当前物品 i
            // f[j - v[i]] + w[i] 表示选择当前物品 i，使用剩余容量 j-v[i] 的最优解加上当前物品价值
            f[j] = max(f[j], f[j - v[i]] + w[i]);
        }
    }
    
    // 输出背包容量为 m 时的最大价值
    cout << f[m] << endl;
}
```
