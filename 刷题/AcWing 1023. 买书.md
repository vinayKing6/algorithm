# 题目

[1023. 买书 - AcWing题库](https://www.acwing.com/problem/content/1025/)

小明手里有n元钱全部用来买书，书的价格为10元，20元，50元，100元。

问小明有多少种买书方案？（每种书可购买多本）

#### 输入格式

一个整数 n，代表总共钱数。

#### 输出格式

一个整数，代表选择方案种数。

#### 数据范围

0≤n≤1000

#### 输入样例1：

```
20
```

#### 输出样例1：

```
2
```

#### 输入样例2：

```
15
```

#### 输出样例2：

```
0
```

#### 输入样例3：

```
0
```

#### 输出样例3：

```
1
```

# 思路

复习：[[AcWing 3. 完全背包问题]]，但这道题是求方案数，先复习一下[[AcWing 278. 数字组合]]

初始状态：`f[0][0]`
目标状态：`f[n][m]`

# 代码
```c++
#include<bits/stdc++.h>
using namespace std;

// N=5 是因为有4种面额（索引从1开始）；M=1010 是金额的最大上限
const int N=5, M=1010;
int n; // 目标总金额
int prices[N], f[M]; // prices 存储面额，f 存储达到对应金额的方案数

int main(){
    // 输入目标金额 n
    cin >> n;

    // 初始化 4 种货币的面额
    prices[1] = 10;
    prices[2] = 20;
    prices[3] = 50;
    prices[4] = 100;
    
    // 【动态规划初始化】
    // f[j] 表示凑出金额为 j 的方案数。
    // 当金额为 0 时，只有一种方案：即什么都不选。
    f[0] = 1;

    // 【核心动态规划逻辑：完全背包求方案数】
    // 遍历每一种面额
    for(int i = 1; i <= 4; i++){
        // 遍历金额，从当前面额 prices[i] 开始到目标金额 n
        // 因为每种硬币可以使用无限次，所以采用正序遍历
        for(int j = prices[i]; j <= n; j++){
            /* 状态转移方程：
               凑成金额 j 的新方案数 = 原有的方案数 + 使用了当前面额 prices[i] 后的新方案数
               即：f[j] = f[j] + f[j - prices[i]]
            */
            f[j] += f[j - prices[i]];
        }
    }

    // 输出凑成金额 n 的总方案数
    cout << f[n] << endl;

    return 0;
}
```
