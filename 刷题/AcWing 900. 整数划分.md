# 题目

https://www.acwing.com/problem/content/902/

一个正整数 n 可以表示成若干个正整数之和，形如：$n=n_1+n_2+\dots+n_k$，其中 $n_1≥n_2≥\dots≥n_k,k≥1$。

我们将这样的一种表示称为正整数 n 的一种划分。

现在给定一个正整数 n，请你求出 n 共有多少种不同的划分方法。

#### 输入格式

共一行，包含一个整数 n。

#### 输出格式

共一行，包含一个整数，表示总划分数量。

由于答案可能很大，输出结果请对 $10^9+7$ 取模。

#### 数据范围

1≤n≤1000

#### 输入样例:

```
5
```

#### 输出样例：

```
7
```

# 思路

先复习一下完全背包问题 [[AcWing 3. 完全背包问题]]

**思路**：把1,2,3, … n分别看做n个物体的体积，这n个物体均无使用次数限制，问恰好能装满总体积为n的背包的**总方案数**（完全背包问题变形）

**初值问题**：  
求最大值时，当都不选时，价值显然是 0  
而求方案数时，当都不选时，方案数是 1（即前 i 个物品都不选的情况也是一种方案），所以需要初始化为 1  
即：`for (int i = 0; i <= n; i ++) f[i][0] = 1;`  
等价变形后： `f[0] = 1`,即什么都不选总和为0的方案为1种

**状态计算**：

$f[i][j]$ 表示前i个整数（1,2…,i）恰好拼成j的方案数  
求方案数：把集合选0个i，1个i，2个i，…全部加起来  
`f[i][j] = f[i - 1][j] + f[i - 1][j - i] + f[i - 1][j - 2 * i] + ...;`  
`f[i][j - i] = f[i - 1][j - i] + f[i - 1][j - 2 * i] + ...;`  
因此 $f[i][j]=f[i−1][j]+f[i][j−i]$; (这一步类似完全背包的推导）

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1010;
int n, dp[N];  // dp数组：dp[j]表示正整数j能被划分为1~n之间数的方案总数
int mod = 1e9 + 7;

int main() {
    cin >> n;
    
    // dp数组初始化：
    dp[0] = 1;// 一个重要技巧：和为0的方案数是1（什么都不选的方案视为1种）。
    
    /*
    外层循环枚举"物品"的价值（从数字1枚举到数字n）；
    将这n看作背包容量，"集合中的数字"(1,...,i)看作所选物品的重量；
    
    这种做法实质是将整数的正整数划分看作是"装满重量为数字n的物品的选择方案"的组合；
    每个数字可以无限次使用（完全背包性质）。
    */
    for(int i = 1; i <= n; i++) {
        /*
        内层循环：随着背包容量（j）递增更新dp；
        变化方向(j从i到n)体现了完全背包的优化：允许某种价值（数字）重复多次；
        递推公式：dp[j] = dp[j] + dp[j-i];
    
        含义：当前数字j的方案数 = 上个阶段第j项（不含数字i的方案） + 包含数字i的方案（即j-i的方案数）。
        */
        for(int j = i; j <= n; j++) {
            dp[j] = (dp[j] + dp[j - i]) % mod;
        }
    }

    /*
    样例验证：整数5的拆分
    正整数5的拆分方案总计包括：
     
    解法共7项：
     
    枚举过程示例：
    1) 1+1+1+1+1
    2) 1+1+1+2
    3) 1+1+3
    4) 1+2+2
    5) 1+4
    6) 2+3
    7) 5
    */
    cout << dp[n] << endl;
    
    return 0;
}
```
