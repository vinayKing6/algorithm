# 题目

https://www.acwing.com/problem/content/1143/

某个局域网内有 n 台计算机和 k 条 **双向** 网线，计算机的编号是 1∼n。由于搭建局域网时工作人员的疏忽，现在局域网内的连接形成了回路，我们知道如果局域网形成回路那么数据将不停的在回路内传输，造成网络卡的现象。

**注意**：

- 对于某一个连接，虽然它是双向的，但我们不将其当做回路。本题中所描述的回路至少要包含两条不同的连接。
- 两台计算机之间最多只会存在一条连接。
- 不存在一条连接，它所连接的两端是同一台计算机。

因为连接计算机的网线本身不同，所以有一些连线不是很畅通，我们用 f(i,j) 表示 i,j 之间连接的畅通程度，f(i,j) 值越小表示 i,j 之间连接越通畅。

现在我们需要解决回路问题，我们将除去一些连线，使得网络中没有回路且不影响连通性（即如果之前某两个点是连通的，去完之后也必须是连通的），并且被除去网线的 Σf(i,j) 最大，请求出这个最大值。

#### 输入格式

第一行两个正整数 n,k。

接下来的 k 行每行三个正整数 i,j,m 表示 i,j 两台计算机之间有网线联通，通畅程度为 m。

#### 输出格式

一个正整数，表示被除去网线的 Σf(i,j) 的最大值。

#### 数据范围

1≤n≤100  
0≤k≤200  
1≤f(i,j)≤1000

#### 输入样例：

```
5 5
1 2 8
1 3 1
1 5 3
2 4 5
3 4 2
```

#### 输出样例：

```
8
```

# 思路

模板题[[AcWing 859. Kruskal算法求最小生成树]]

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 210; // 定义点的数量上限
int n, p[N], ans, sum, k; // n: 点数, k: 边数, sum: 所有边权之和, ans: MST 边权之和

// 并查集：查找函数，带有路径压缩功能
int find(int x){
    if(p[x] != x){
        p[x] = find(p[x]); // 将当前节点直接指向根节点，加速查找
    }
    return p[x];
}

// 定义边结构体
struct Edge{
    int a, b, c; // a, b 为顶点，c 为边权
};

Edge e[N*2]; // 存储所有的边

// 排序规则：按边权从小到大排序
bool cmp(Edge e1, Edge e2){
    return e1.c < e2.c;
}

// Kruskal 算法主体
void kruskal(){
    for(int i = 0; i < k; i++){
        int a = e[i].a;
        int b = e[i].b;
        int c = e[i].c;
        
        // 分别查找两个顶点的根节点
        int pa = find(a);
        int pb = find(b);
        
        // 如果两个顶点不在同一个集合（不构成环），则将该边加入生成树
        if(pa != pb){
            ans += c;       // 累加 MST 的总权值
            p[pa] = pb;     // 合并两个集合
        }
    }
}

int main(){
    cin >> n >> k;
    
    // 初始化并查集：每个点最初都是自己的根节点
    for(int i = 1; i <= n; i++)
        p[i] = i;
    
    // 读入所有边并计算所有边的总权值 sum
    for(int i = 0; i < k; i++){
        int a, b, c;
        cin >> a >> b >> c;
        e[i] = {a, b, c};
        sum += c; 
    }
    
    // 1. 将所有边按权重排序
    sort(e, e + k, cmp);
    
    // 2. 执行 Kruskal 算法求最小生成树
    kruskal();
    
    // 3. 输出结果：总权值减去最小生成树权值 = 舍弃的冗余边权值之和
    cout << sum - ans << endl;
}
```
