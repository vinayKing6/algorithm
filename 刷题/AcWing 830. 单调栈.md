# 题目

https://www.acwing.com/problem/content/832/

给定一个长度为 N 的整数数列，输出每个数左边第一个比它小的数，如果不存在则输出 −1。

#### 输入格式

第一行包含整数 N，表示数列长度。

第二行包含 N 个整数，表示整数数列。

#### 输出格式

共一行，包含 N 个整数，其中第 i 个数表示第 i 个数的左边第一个比它小的数，如果不存在则输出 −1。

#### 数据范围

1≤N≤$10^5$  
1≤数列中元素≤$10^9$

#### 输入样例：

```
5
3 4 2 7 5
```

#### 输出样例：

```
-1 3 -1 2 2
```

# 思路
这道题是 **单调栈经典题：求每个数左边第一个比它小的数（Nearest Smaller to Left）**。  
我一步一步给你讲 **题意 → 思路 → 手动模拟 → 为什么是单调栈**。

---

## 一、题目意思

给一个数组：

```
a1 a2 a3 ... an
```

对每个数 `ai`：

找到

```
左边第一个 < ai 的数
```

如果没有：

```
输出 -1
```

---

### 示例

输入

```
5
3 4 2 7 5
```

输出

```
-1 3 -1 2 2
```

解释：

|当前数|左边|第一个更小|
|---|---|---|
|3|无|-1|
|4|3|3|
|2|3 4|无|
|7|3 4 2|2|
|5|3 4 2 7|2|

---

## 二、核心思想（单调栈）

维护一个 **单调递增栈**

栈特点：

```
栈底 → 栈顶
从小到大
```

例如：

```
2
3
5
8   ← 栈顶
```

---

## 三、算法步骤

每来一个新数 `x`：

### ① 弹出不可能成为答案的数

```
while (栈不空 && 栈顶 >= x)
    pop
```

因为：

```
栈顶 >= x
```

说明：

```
栈顶不可能成为任何未来元素的
左边更小值
```

所以直接删。

---

### ② 判断答案

如果：

```
栈空
```

说明：

```
左边没有更小
```

输出：

```
-1
```

否则：

```
栈顶就是左边第一个更小值
```

---

### ③ 当前元素入栈

```
push(x)
```

因为它可能成为未来元素的答案。

---

## 六、为什么时间复杂度是 O(n)

每个元素：

```
最多入栈一次
最多出栈一次
```

所以：

```
总操作 ≤ 2n
```

时间复杂度：

```
O(n)
```

# 代码

```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 1e5 + 10;

int a[N];   // 存储输入数组
int n;      // 数组长度

stack<int> s;  // 单调栈，用来维护左侧元素

int main(){

    cin >> n;  // 输入元素个数

    // 读入数组
    for(int i = 1; i <= n; i++) 
        cin >> a[i];

    // 遍历数组
    for(int i = 1; i <= n; i++){

        // 维护单调递增栈
        // 如果栈顶元素 >= 当前元素
        // 说明栈顶不可能成为左侧第一个更小元素
        // 因此弹出栈顶
        while(s.size() && s.top() >= a[i])
            s.pop();

        // 如果栈为空
        // 说明左侧没有比当前元素更小的数
        if(!s.size())
            cout << -1 << " ";

        // 如果栈不为空
        // 栈顶元素就是左侧第一个比当前元素小的数
        else
            cout << s.top() << " ";

        // 当前元素入栈
        // 可能成为后面元素的“左侧第一个更小值”
        s.push(a[i]);
    }
}
```
